# 時代区分

現時点では、機構世界全体の厳密な年代表は未確定である。
このページは、記事間で基準時点を揃えるための `仮時代区分` と `運用ラベル` をまとめる。

## 大時代ラベル

### 神話時代

- 神、魔女、世界モデルの起源に近い時代
- 神話、原初設定、宗教的伝承を置くときに使う

### 古代

- 古層文明、失われた技術、初期国家、古い宗教体系など

### 前近代

- 現代星間文明の直前までを広く含む中間ラベル
- 細分化前の暫定運用向け

### 近代

- 魔導理論の定量化、再現魔術の体系化、近代国家化、工業化が進む時代

### 現代

- 星間文明、ISMO、星間国家連合、星間通信網が成立している現在基準の時代

## 補助ラベル

### 星間進出前 / 星間進出後

- 惑星内文明か、星間文明かを区切るための補助ラベル

### ISMO成立前 / ISMO成立後

- ISMOの成立が設定上重要な境目になるための補助ラベル

### エリミネーター公知前 / エリミネーター公知後

- 一般社会や上層部における認知状態の変化を扱う時に使う

## 物語進行ラベル

主要物語が固まるまでは、以下の相対ラベルで運用する。

- `物語開始前`
- `物語開始時`
- `第一部後`
- `第二部後`
- `終盤`

## 運用ルール

- 年号が未確定でも、時代ラベルだけで記事は書いてよい
- `現代` だけで不足する場合は、補助ラベルを併記する
- 物語が固まったら、相対ラベルを出来事ベースや年号ベースへ置換する

## 例

```md
- 基準時点: 現代 / ISMO成立後
```

```md
- 基準時点: 物語開始時 / 現代
```
