# 世界観概略

機構世界の入口用メモ。  
ここでは `世界の骨格` `現代文明の構図` `物語でよく使う軸` だけを短くまとめる。  
詳細な整理や議論の蓄積は [世界観統合メモ](世界観統合メモ.md) を参照。

## 一文要約

魔術が神秘として残りつつも魔導理論と魔導工学によって体系化され、複数の星間文明が並立する宇宙で、超国家的危機対応組織 ISMO が、エリミネーターと呼ばれる宇宙規模の破滅意志に対抗している世界。

## 世界の骨格

- 知性体の住む惑星は比較的多く、単一起源ではない複数の文明が星間航行に到達している
- 星間文明は存在するが、宇宙全体を統一する単一政府はなく、多中心的な政治秩序が続いている
- 惑星ごとの文化差、魔術理論差、制度差は大きく、連合加盟圏と非加盟圏の差も強い
- 魔術は消えていないが、現代社会では `再現魔術` と `魔導工学` が主流である
- その一方で、純粋魔術、魔法、魔女、神のような形而上的存在も世界の深層には実在する

## 現代文明の姿

- 宇宙ではエーテルを基盤にした標準化魔導技術が広く使われる
- 航路、寄港、補給、通信、検疫は [星間国家連合](..\02_組織・国家・勢力事典\国家・政府\星間国家連合.md) の条約網でかなり整理されている
- ただし連合の外側には、非加盟圏、半接続交易圏、危険宙域、在来文明保護圏が残っている
- 現場の秩序維持、救難、封鎖、初動危機対応を最もよく担うのは [星間治安維持機構（ISMO）](..\02_組織・国家・勢力事典\軍事組織\星間治安維持機構（ISMO）.md) である

## 魔術体系の超要約

- `純粋な魔術`
  地域、継承、神秘、術者個人の才能に強く依存する古典的体系
- `再現魔術`
  工学的手法で魔術現象を標準化・再利用する現代の主流体系
- `魔法`
  現象を起こす技術ではなく、世界モデルや法そのものに近い上位の力

## 魔導理論の超要約

- 魔力は単一ではなく、複数の相と第一質料を持つ
- 宇宙実務では `エーテル` が最も重要で、星間技術、通信、再現魔術、ISMO装備の基盤になっている
- 深層理論では、マナ、アストラル、オド、虚無のような第一質料と世界モデルが重なって、現在の世界が成立している
- `魔法使い` は学術的に魔法へ到達した者、`魔女` は魂が世界を構成するに至った者であり、単純な上位下位ではない

## 種族と人格主体の要約

- 人間以外にも、外見差が小さい `基本種族` が複数存在する
- 一方で、非基本種、人工知性、魔導知性のように、法的な扱いが揺れやすい存在も多い
- この世界では `魂を持つこと` と `法的に人格主体として認められること` は別問題であり、そこに制度上の緊張がある

## 政治地図の要約

- 連合は超国家政府ではなく、国家間条約共同体である
- 強国は重いが、単独で全てを止める露骨な拒否権国家クラブではない
- 主要加盟主体として、次の四極が現在の基準になっている
  - [アウレリア共同圏](..\02_組織・国家・勢力事典\国家・政府\アウレリア共同圏.md)
    交易、港湾、物流、金融
  - [ルーメン技術連邦](..\02_組織・国家・勢力事典\国家・政府\ルーメン技術連邦.md)
    規格、工業、通信、技術認証
  - [ノルトヴェルト帝国](..\02_組織・国家・勢力事典\国家・政府\ノルトヴェルト帝国.md)
    最大級の単独宇宙帝国、秩序と主権と規模
  - [東天諸邦連盟](..\02_組織・国家・勢力事典\国家・政府\東天諸邦連盟.md)
    地域多様性、伝統魔術、文化圏としての重さ

## ISMOの位置付け

- ISMOは国家ではない
- だが艦隊、執行官、法制度、福利、独自身分を持つため、単なる民間組織でもない
- 役割は `戦争遂行` ではなく、治安維持、救難、調停、封鎖、異常事態対応、エリミネーター対処である
- 戦力はかなり高いが、艦そのものよりも `エキスパートな執行官を通すための艦隊` という性格が強い
- 建前と実力がずれている、意図的に歪な組織であることが、この世界での大きな特徴になっている

## アリナの位置付け

- [アリナ](..\01_人物事典\現代人物\アリナ.md) は、ISMOの中枢を支える人工知性 / 魔導知性 / 人工魔術原理である
- ただの便利AIではなく、人格と意思を持つ
- 建前上はISMO資産に近く扱われる場面もあるが、実態としては `本人の協力によってISMOが成立している` 側面が強い
- 彼女の存在は、ISMOの技術優位と制度的危うさの両方を支えている

## エリミネーターの要約

- エリミネーターの本質は、宇宙自身の根源的破滅意志の外在化である
- 目的は、宇宙、構造、文明、生命の破壊
- 直接破壊、汚染、精神侵略、擬態、災害化など多様な形を取る
- 対抗にはエーテルがほぼ必須であり、通常戦力だけでは土俵に立てない場面が多い
- 人類史上の公的な最初の大災厄は [ミレイア災厄](..\04_歴史・年表\戦争・事件\ミレイア災厄.md) として記憶されている

## 現代の傷跡

- [アルトリス星系](..\03_天体・星系・宙域事典\星系\アルトリス星系.md) と [ミレイア](..\03_天体・星系・宙域事典\惑星・衛星\ミレイア.md) は、一世紀を経ても傷跡が残る災厄の象徴である
- 公的には未確認生物災害として語られるが、ISMO上層の最重要機密では別の真相が共有されている
- [ディアーナ教団](..\02_組織・国家・勢力事典\宗派・学術機関\ディアーナ教団.md) のような、破滅思想を独自解釈した勢力も現代の不安定要因になっている

## 物語の足場

- 主人公は [カイ・レンフォード](..\01_人物事典\現代人物\カイ・レンフォード.md)
- 出発点は非加盟の辺境コロニー [外縁コロニー・ラスティア](..\03_天体・星系・宙域事典\宇宙港・コロニー・人工天体\外縁コロニー・ラスティア.md)
- 彼は事件をきっかけに ISMO の `特任編入枠` で拾われる
- 初期ホームは [第七機動群（アルシオン機動群）](..\02_組織・国家・勢力事典\軍事組織\第七機動群（アルシオン機動群）.md) と、その母艦 [アルシオン](..\06_艦船・兵器・装備・遺物事典\艦船\アルシオン.md)
- 平時の休暇や補給の戻り先としては、A級協定港 [港環都市カルディア](..\03_天体・星系・宙域事典\宇宙港・コロニー・人工天体\港環都市カルディア.md) が機能する

## 現代の主要対立

- 連合秩序と非加盟圏のあいだの摩擦
- 標準化された再現魔術と、地域固有の純粋魔術の摩擦
- 人工知性や非人類をどこまで人格主体として扱うか
- ISMOの必要性と危険性
- 破滅を防ぐための力が、同時に別の歪みを生むこと
