# 連合主要国ラフ案

## 管理メタ

- 基準時点: 現代
- 公知度: 作者用
- 主な認知層: 作者
- 真偽: 暫定案
- 変動性: 高
- 時点依存: 中

## 方針

- まずは `中核加盟国枠` に入りやすい国家を4つ置く
- 個別本記事を書く前の政治・機能ラフとして使う
- 現時点では `連合制度の圧力源として何を担うか` を優先し、細部の歴史や主要人物は後回しにする

## 暫定候補

### 1. アウレリア共同圏

- 立ち位置:
  交易、金融、共同航路、協定港ネットワークを握る海運商業国家群の連合体
- 連合内での役割:
  予算拠出、補給基盤、協定港提供、共同航路維持の面で重い
- 政治姿勢:
  連合賛成寄りだが、理念よりも `交易の安定` を重視する現実派
- ISMOとの関係:
  航路保全と救難では協力的
  ただし通商が止まる強引な封鎖や長期作戦には厳しい
- 物語上の使いどころ:
  連合の金庫番
  `中立` を掲げつつ利益で秤を傾けるタイプ

### 2. ルーメン技術連邦

- 立ち位置:
  再現魔術工業、通信、標準規格、艦載機関、資格制度に強い技術官僚国家
- 連合内での役割:
  共同基盤の規格制定、通信網、資格相互承認、共同技術審査で強い
- 政治姿勢:
  制度と標準化を重んじる連合推進派
  ただし技術優位を失う改革には慎重
- ISMOとの関係:
  技術的には近いが、アリナ問題や秘匿技術では緊張を抱える
- 物語上の使いどころ:
  `正しさ` と `標準化` を武器に政治を動かす国

### 3. ノルトヴェルト帝国

- 立ち位置:
  単独主権国家としては最大級の領域、人口、艦隊を抱える宇宙帝国
- 連合内での役割:
  危機対応能力、軍事的抑止力、星系規模の統治実績を理由に、執行評議会で最大級の発言力を持つ
- 政治姿勢:
  連合そのものは利用するが、主権制限や介入基準には強く反発する
- ISMOとの関係:
  必要時には頼るが、領域内介入や監査には不快感を隠さない
- 物語上の使いどころ:
  `露骨な拒否権はないが、実質的に止めてくる最大級の帝国`
  連合を内部から歪める圧力の象徴

### 4. 東天諸邦連盟

- 立ち位置:
  多数の星系国家・自治圏・伝統魔術文化圏から成る緩やかな連盟
- 連合内での役割:
  圏域代表性、文化多様性、主権尊重、在地魔術の保護を主張する
- 政治姿勢:
  連合参加には積極的だが、中央集権化や一律規格には警戒的
- ISMOとの関係:
  辺境救難や広域安全保障では評価する一方、地域固有理論や在来秩序への干渉を嫌う
- 物語上の使いどころ:
  `連合反対` ではなく `連合のやり方にうるさい` 側
  伝統魔術と再現魔術の摩擦も載せやすい

## 4国のバランス

- アウレリア共同圏:
  金、港、航路
- ルーメン技術連邦:
  規格、通信、技術
- ノルトヴェルト帝国:
  最大規模の単独帝国、艦隊、威圧、主権圧力
- 東天諸邦連盟:
  文化、地域代表、伝統理論

この4枠があると、星間国家連合は `金と物流` `技術と制度` `軍事と主権` `地域多様性と伝統` の4本で引っ張られる構図になる。

特にノルトヴェルト帝国は、単一国家としては最大級であるため、制度上の拒否権がなくても `実質的に無視しづらい圧力源` として振る舞う。連合があえて常任拒否権を設けない思想を取る場合、その設計は半ばこの帝国を一国で制度停止させないための工夫でもある。
