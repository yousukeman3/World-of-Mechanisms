# 知識レイヤー一覧

機構世界の記事で `誰がその情報を知っているか` を管理するための運用メモ。
個人単位ではなく、まずは `認知層` と `公知度` で管理する。

## 基本方針

- まず `主な認知層` を集団で書く
- 個別の例外は `例外的認知者` で補足する
- 本文中で `通説` と `真相` を混ぜない
- 真相を知っていても、世界内の大半が知らないなら `公知度` は上げない

## 公知度ラベル

### 一般公知

- 一般社会で広く知られている
- 専門教育や所属を問わず話題にできる

### 限定公知

- 一般人にも断片的には知られている
- ただし噂、伝承、通俗化された知識に留まることが多い

### 専門家公知

- 学者、魔術師、技術者、軍人などの専門層では知られている
- 一般社会では常識ではない

### 組織内限定

- 特定組織の内部では共有されている
- ただし部局や階級によって理解度に差がある

### 組織機密

- 特定組織の中でも限られた者しか知らない
- 漏洩時に政治、軍事、社会秩序へ大きな影響が出る

### 国家上層限定

- 国家や連合の上層部、監督機関、限られた実務者のみが知る
- 公知ではないが、単一組織の内部機密より広い

### 真相

- 作者視点での確定情報
- 世界内ではほぼ知られていない、または到達困難

## 認知層の書き方

記事の `主な認知層` には、以下のような粒度で複数記入してよい。

- 一般市民
- 惑星文明の魔術師
- 再現魔術士
- 呪術師
- 魔法使い
- 魔女
- 国家上層部
- 星間国家連合監督部門
- ISMO一般構成員
- ISMO上層部
- ISMO技術研究開発局
- 特定宗派
- 学界

## 例外的認知者

主流の認知層から外れる個人や少数集団がいる場合だけ補足する。

例:

- `例外的認知者: アリナ / 最高司令官 / 某宗派の司祭長`

## 真偽ラベル

### 通説

- 世界内で一般的に信じられている説明
- 真相と一致しているとは限らない

### 有力説

- 専門家層で有力だが、未確定

### 論争中

- 学派、国家、宗派で見解が割れている

### 機密情報

- 世界内で一部には知られているが、公開されていない

### 真相

- 作者視点での確定情報

### 真相混在

- 記事内に通説と真相が併記されている
- この場合は本文の見出しで必ず層分けする

## 記事への入れ方

最低限、記事冒頭の `管理メタ` に以下を入れる。

```md
## 管理メタ

- 基準時点:
- 公知度:
- 主な認知層:
- 真偽:
- 変動性:
- 時点依存:
- 例外的認知者:
```

## 本文の推奨構成

認知差が大きい記事では、本文も以下のように分けると扱いやすい。

```md
## 世界内の通説
## 専門家の理解
## 機密情報
## 真相
```

## 運用上の注意

- `一般に知られていない` と `誰にも知られていない` は別
- `真相` を書く記事では、通説との差分を明示する
- 人物記事でも、本人の自己認識と社会の認識が違う場合は分けて書く
