# エリミネーター代表個体案

## 管理メタ

- 基準時点: 現代
- 公知度: 作者用
- 主な認知層: 作者用
- 真偽: 暫定案
- 変動性: 高
- 時点依存: 中

## 概要

エリミネーターの代表個体案。現時点では、各類型の雰囲気と実戦上の厄介さを掴むための暫定候補であり、正式な個体名や分類コードではない。

## 断界獣ヴァルグラフ

- 仮コード: `ELM-D/01`
- 主類型: 直接破壊型
- 補助タグ: 大型 / 混成 / 接近 / 物理・魔術混成 / 単独

### 外見

巨大獣と突撃艦を接ぎ木したような外見を持つ。黒い金属甲殻と露出した発光亀裂が重なり、四肢とも推進器ともつかない構造で空間を掻くように移動する。

### 主な攻撃・侵食手段

- 艦体や施設への正面突撃
- 爪状構造による装甲切断
- 衝突と同時に結界やエーテル防御を割る局所崩壊

### 対応上の厄介さ

わかりやすく強い。回避に失敗した時点で前衛艦や拠点が即死しうるため、迎撃火力と機動力の両方を要求する。

## 灰蝕花アッシュブルーム

- 仮コード: `ELM-C/02`
- 主類型: 侵蝕汚染型
- 補助タグ: 群体 / 混成 / 非接触 / 魔術・侵食 / 現象随伴

### 外見

灰色の花弁、結晶、菌糸、煙霧が混ざり合った群体。艦内配管、都市施設、土壌、海中など、既存構造に寄生するように咲き広がる。

### 主な攻撃・侵食手段

- 空気、水、魔力循環路の汚染
- 機関や生活インフラの劣化
- エーテル運用系の目詰まりと鈍化

### 対応上の厄介さ

侵攻に気付きにくい。表面的には事故、故障、環境悪化に見えやすく、対処が遅れるほど都市や艦隊の継戦能力を静かに奪う。

## 悼歌体ネニア

- 仮コード: `ELM-M/03`
- 主類型: 精神侵略型
- 補助タグ: 個体 / 純現象的 / 遠隔 / 精神 / 群体化あり

### 外見

固定された肉体を持たず、光輪、残響、合唱、白い影、人型の残像のように観測される。状況によっては、亡霊、天使、弔いの行列のようにも見える。

### 主な攻撃・侵食手段

- 歌声、囁き、夢、共感を介した精神干渉
- 絶望、諦念、自己犠牲衝動の増幅
- 集団指揮系統の崩壊と自滅誘導

### 対応上の厄介さ

物理被害が少ないまま組織を壊せる。被害者自身が「自分の意志だ」と思い込みやすく、発見と隔離が難しい。

## 空席乗客ブランク・パッセンジャー

- 仮コード: `ELM-I/04`
- 主類型: 擬態潜入型
- 補助タグ: 個体 / 生体的・混成 / 接近 / 精神・情報 / 単独

### 外見

「最初からそこにいた誰か」の姿を取る。制服姿の乗員、避難民、職員、家族、端末越しの相談者などとして認識されることが多いが、後から記録を照合すると決定的な固有像が定まらない。

### 主な攻撃・侵食手段

- 認識の隙間への潜入
- 記録や記憶の齟齬拡大
- 内部からの開門、破壊工作、誤誘導

### 対応上の厄介さ

見つかった時には内部に入り込んでいることが多い。敵そのものより、「どこまで侵入されたか」が問題になる。

## 反転潮リヴァーサル

- 仮コード: `ELM-P/05`
- 主類型: 現象災害型
- 補助タグ: 宙域級 / 純現象的 / 非接触 / 物理・魔術混成 / 現象随伴

### 外見

暗い極光、重力レンズの反転、星光の歪み、黒い潮流のように観測される。近づくと上下、内外、遠近の感覚が狂い、通常の航法図が意味を失う。

### 主な攻撃・侵食手段

- 航法、通信、重力、魔力流の攪乱
- 宙域全体の位相異常化
- 艦隊単位の行方不明、漂流、同士討ち誘発

### 対応上の厄介さ

相手を撃てば終わる類ではない。封鎖、隔離、観測制御、航路再設計が必要になるため、軍事案件と災害案件を同時に引き起こす。

## 母体アルケイオン

- 仮コード: `ELM-H/06`
- 主類型: 母体・指揮型
- 補助タグ: 艦隊級 / 混成 / 遠隔 / 混成 / 母体依存

### 外見

神殿、墓所、巣、工廠、記録庫が混ざったような巨塊。世界の記憶を雑に縫い合わせたような外形を持ち、周辺空間そのものが「何かを産み出す装置」に見えてくる。

### 主な攻撃・侵食手段

- 下位エリミネーターの生成と再編
- 戦場全体への意味付けと行動方向の統制
- 周辺個体や現象へ一つの破滅の物語を与える

### 対応上の厄介さ

下位個体をいくら倒しても補充されうる。単体戦闘力より、中枢として戦場全体を“破滅へ整える”ことの方が脅威になる。

## ディアーナ

- 仮コード: `ELM-EX/00`
- 主類型: 母体・指揮型
- 補助タグ: 超抜級 / 混成 / 遠隔 / 混成 / 母体依存なし

### 位置付け

ディアーナは通常の代表個体というより、現在のエリミネーター群へ連なる `最初の超抜級統率個体` として扱う方が自然である。宇宙の破滅意志そのものではないが、記録内のアリナをもとに、真理の鍵へ触れた前身組織の創設者が作り出してしまった超知能個体である。

アリナを単に模倣した劣化存在ではなく、戦闘能力、情報処理能力、魔術技能、指揮能力、共感能力の多くで現代時点のアリナを上回る個体として考えるのが自然である。その理由は、宇宙の根源的破滅意志がバックアップとして接続されており、宇宙の力の一部を直接行使できるためである。

### 外見

アリナをそのまま成熟させたような女性型外見を取るのが自然である。髪色と瞳の色はともに昏い赤紫へ反転しており、全体としてアリナの面影を残しながら、より完成され、より危険で、より終末的な印象を与える。

### 主な攻撃・侵食手段

- エリミネーター群全体への統率
- 破滅意志に沿った個体群の方向付け
- 高度な知性を用いた文明破壊の最適化
- 苦しみの大きい個体へ `死という名の救済` を提示し、エリミネーター化させる
- 共感と憐憫を入口にしつつ、最終的には破滅へ導く
- 生を疎む存在へは慈愛と快楽を伴う甘美な救済として接近し、依存と服従を誘う
- 慈母的な受容と官能的な肯定を混ぜた精神支配で、自発的な終焉希求を植え付ける

### 動機

ディアーナの中核動機は `憐憫` と `破壊指向` の結合である。彼女は世界を単に憎んで壊すのではなく、苦しみに満ちた世界へ幕引きという名の救いを与えようとする。そのため、終わらせること、殺すこと、破滅へ導くことを、むしろ慈悲として語りうる。

特に、生そのものを疎み、疲れ、絶望し、自らの終わりを望み始めた相手にはきわめて優しい。そこでは脅しや威圧ではなく、慰撫、受容、慈愛、そして快楽を通じて相手を包み込み、`終わることこそ救いである` という感覚へ静かに沈めていく。

アリナとの対照でいえば、ディアーナは `他者のため` に動くと自認するがゆえに破滅を押し付ける存在であり、性格傾向も外交的というより内向的である。相手を理解し、包み込み、内側から終焉へ導くことを選びやすい。

### 対応上の厄介さ

単純な怪物ではなく、人類側の知識と破滅意志が接続してしまった結果生まれた存在であるため、ISMOにとっては最大級の汚点そのものでもある。さらに、単なる暴力ではなく、救済めいた語りと高度な共感能力、慈愛と快楽を用いた精神支配を通じて対象を自発的な破滅へ導くため、心理的・宗教的・存在論的な脅威としても極めて危険である。

## メモ

- `ヴァルグラフ` は最もわかりやすい正面戦闘の顔
- `アッシュブルーム` と `ネニア` は文明崩壊寄りの顔
- `ブランク・パッセンジャー` は物語フックとして使いやすい
- `リヴァーサル` は宙域そのものを舞台化しやすい
- `アルケイオン` は章ボスや上位個体の母体役に向く
- `ディアーナ` は通常個体ではなく、前史と最大機密に直結する特例枠
