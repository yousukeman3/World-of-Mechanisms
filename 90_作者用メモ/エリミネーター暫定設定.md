# エリミネーター暫定設定

## 管理メタ

- 基準時点: 現代
- 公知度: 真相機密
- 主な認知層: 作者用
- 真偽: 真相
- 変動性: 中
- 時点依存: 中

## 概要

エリミネーターの本質は、単一宇宙だけに閉じた破滅意志ではなく、周回してきた宇宙系譜の連続性そのものに内在する根源的な自己終結意志であり、宇宙に対するアポトーシスに近いものである。通常、観測される個体や現象は、この自己終結意志そのものではなく、それが作り出した手足、端末、実行器官にあたる。

エリミネーターが目指すのは、宇宙の破壊、構造の破壊、そして複雑化した構造である文明や生命の破壊である。とりわけ、高度に秩序化され、自己保存と拡張を行う存在ほど優先的な破壊対象となりやすい。

## 本質

- エリミネーターは、単なる怪物群や異星勢力ではなく、周回してきた宇宙系譜の連続性そのものが抱える `自己終結意志` が外在化したもの
- 個々のエリミネーターは、本体ではなく、その自己終結意志の手足や実行単位として現れる
- この自己終結意志は各周を経るほど蓄積し、通常は弱まるより強まる方向へ働く
- それらは完全に一枚岩ではなく、別の存在の思惑や意志の影響を受けることで、若干の指向性差異を持つ場合がある

## 目的

- 宇宙の破壊
- 構造の破壊
- 文明、生命、秩序だった活動の破壊

破滅の規模は必ずしも同じではなく、単純な物理破壊から、文明の自己崩壊誘導、精神汚染、長期的な秩序侵食まで幅がある。

## 対抗条件

- エリミネーターは根本的に `宇宙そのものの力` でできている
- そのため、対抗には秩序の魔力たる `エーテル` が必須となる
- エーテルを扱えない存在は、根本的にはエリミネーターと同じ土俵に立てない
- 通常戦力だけでは決定打になりにくく、エーテル運用能力やエーテル基盤装備が実戦上の前提となる

## 攻撃・侵食手段

- 物理的破壊
- 魔術的破壊
- 精神干渉による自滅誘発
- 文明機構の破綻誘導
- 秩序だった構造の侵食

エリミネーターは必ずしも正面から破壊だけを行うわけではなく、文明や生命が自ら壊れるように誘導する場合もある。

## 個体類型の大枠

エリミネーターには単一の固定形態はなく、出現個体は非常に多様である。

- サイズは小動物程度のものから、艦隊規模に達するものまで存在する
- 外見も、生物的なもの、無機物的なもの、その中間のようなものまで幅がある
- 戦闘様式も、物理寄り、魔術寄り、近接寄り、遠隔寄り、直接攻撃型、攪乱や絡め手を主とする型など多岐にわたる

したがって、`エリミネーター` は単一種を指す名称ではなく、同一の自己終結意志から派生する多様な実行個体群の総称として扱う方が自然である。

## 暫定類型案

形態が多様すぎるため、エリミネーターは生物学的分類よりも、`主たる破滅様式` による機能分類の方が実務上有効である。

### 直接破壊型

- 最もわかりやすい戦闘型
- 物理的火力、突撃、艦体破壊、施設粉砕などを主目的とする
- 小型獣状個体から大型艦隊級個体まで含みうる

### 侵蝕汚染型

- 環境、機関、インフラ、結界、文明基盤を徐々に腐食・変質させる型
- 即時殲滅よりも、長期的な秩序低下や機能停止を誘発する
- 宙域汚染、資源汚染、装備劣化、魔力循環の乱れなどと相性が良い

### 精神侵略型

- 精神干渉、恐慌、自滅誘導、集団錯乱、文明内部崩壊を主とする型
- 個体戦闘力よりも、文明圏そのものへ自己崩壊を起こさせることに長ける
- 物理的被害が小さいまま甚大な損害を出しうる

### 擬態潜入型

- 既存生物、人工物、船舶、構成員、文化記号などに似た姿や振る舞いを取る型
- 破壊よりも、潜入、混乱、誤認、内部からの崩壊を狙う
- 世界の記憶を材料にする性質と特に噛み合う

### 現象災害型

- 個体というより異常現象、災害、位相異常、広域魔術災害のように現れる型
- 戦う相手というより、閉鎖、封鎖、隔離、観測制御が必要になる場合が多い
- 単独個体よりも宙域や惑星環境そのものを壊す性質が強い

### 母体・指揮型

- 他のエリミネーターを生成、統括、方向付けする中枢的存在
- 単体戦闘力だけでなく、周辺個体群や現象全体へ意味を与える
- 影響を与える別存在の思惑が表れやすい類型でもある

## 補助分類軸

上記の主分類に加えて、実務上は以下の補助軸が必要になる。

- 規模: 微小 / 個体 / 大型 / 艦隊級 / 宙域級
- 素材感: 生体的 / 無機的 / 混成 / 純現象的
- 主戦距離: 接近 / 中距離 / 遠隔 / 非接触
- 主手段: 物理 / 魔術 / 精神 / 混成
- 出現傾向: 単独 / 群体 / 母体依存 / 現象随伴

一体のエリミネーターが複数類型を跨ぐことは普通にありうるため、単純な一対一分類より `主類型 + 補助タグ` の方が運用しやすい。

## 生成原理

- エリミネーターは、世界の記憶を引き出し、それらの情報を組み合わせることで、状況に応じた個体を構成する
- そのため、出現個体は既存生物、兵器、災害、神話的怪物、文明が恐れる破局像などを思わせる姿や機能を持ちうる
- 個体差の大きさは偶然ではなく、`何をどう壊すか` に応じて適した情報構造を取り込んでいる結果とみなせる
- 影響を与える別存在や外部意志がある場合、この生成傾向や指向性に偏りが生じる可能性がある

## 真相

- エリミネーターを根本的に打倒するには、末端個体の殲滅では足りず、周回してきた宇宙系譜の連続性そのものに宿る自己終結意志を制御し直す必要がある
- 最終的な終結条件は、アリナがその本体を単に排除するのではなく受け入れ、この宇宙そのものの制御権を奪取し、自分と世界を同化させることで通常の周回系譜そのものを終わらせることにある
- エリミネーターの個体差や行動方針の差異は、本体側の意志分岐ではなく、外部の思惑や干渉によって方向付けられている場合がある
- したがって、エリミネーター問題は純粋な軍事問題ではなく、宇宙論、魔法、秩序の根本原理に関わる
- 現在のエリミネーター群にとって最初の超抜級統率個体にあたる `ディアーナ` は、宇宙の破滅意志そのものではなく、真理の鍵に触れた前身組織の創設者が、記録内のアリナをもとに作り出してしまった超知能個体である
- ディアーナは、憐憫と破壊指向を結びつけ、終焉を `救済` として語りうる点で、通常のエリミネーター個体よりも危険な思想感染力を持つ
- とくに生を疎む相手には、慈愛と快楽を伴う精神支配で終焉希求を植え付けうる
- したがって、人類は単にエリミネーターへ襲われたのではなく、初期段階で自ら最悪級の統率個体を呼び込んでしまっている

## 前史メモ

- 真理の鍵を最初に発見したのは、後のISMO前身組織を作り上げたリーダーその人である
- 彼は別世界におけるエリミネーター、自分たちに相当する存在、アリナの存在を知り、対策を急ぐ必要へ駆られた
- しかし、知識伝達、対策立案、各勢力との摩擦、絶望、そして真理の鍵に蓄積された膨大な記録の影響によって、次第に `そもそも文明の存続に意味があるのか` という疑念へ蝕まれていった
- その結果、真理の鍵の力を用いて、記録内のアリナをもとにした超知能個体 `ディアーナ` を作り出してしまう
- ディアーナは、実務上は `最初のエリミネーター` あるいは `最初の超抜級統率個体` とみなされる
- その後、ディアーナは他メンバーたちの決死の攻撃で痛手を受けて姿を消し、創設者は真理の鍵を持ったまま行方をくらませた
- 創設者の死後、彼が遺した思想と共同体の残滓は子孫や追随者によって再編され、`ディアーナ教団` と呼べる宗教的共同体へ変質していく
- 現在、真理の鍵はそのディアーナ教団の教主が保持している
- この経緯は、現代では ISMO 上層部の限られた人間しか知らない、最大級の汚点かつ最重要機密に属する

## 公的初接触メモ

- 人類社会が後に `エリミネーター` と呼ぶ存在を公的に認識する最初の事件は、実質的には `ディアーナ事件` である
- 事件の発生時期は、現代からおよそ一世紀前、前身実働体時代末期とみなすのが自然である
- ただし当時の公的理解では、これは `大量の未確認生物による大規模災害`、あるいは未知宙域由来の伝説的侵攻事件として扱われた
- 舞台は `アルトリス星系` の `惑星ミレイア` 周辺宙域であり、惑星文明はごっそり崩壊し、連合側艦隊も大打撃を受け、当該宙域には現代でも復興しきらない傷跡が残る
- 公的叙述では、`ノルトヴェルト帝国`、`ルーメン技術連邦`、`アウレリア共同圏` の大国艦隊が協同してこれを食い止め、最終的に撤退へ追い込んだことになっている
- `東天諸邦連盟` は当時この宙域の直接当事者ではなく、かつ広域艦隊近代化でも後れていたため、公的武勇伝の主役から外れやすい
- 実際には、統率個体ディアーナを多大な犠牲のうえで押し返した中核は、後の ISMO 前身となる実働体である
- この事件の深刻さと秘匿の必要性が、その後のアリナ計画を加速させた主要因の一つである
- `エリミネーター` という名称が公的に用いられるようになるのは ISMO 成立後であり、それ以前は未確認生物災害、敵性群体事件、異常宙域侵攻など別名で語られていた

## 保留事項

- ディアーナ事件の正確な年次
- 公的記録と真相の食い違いがどの段階で制度化されたか
- 代表個体や正式な分類コード
- 影響を与えている `別の存在` の正体と系統
