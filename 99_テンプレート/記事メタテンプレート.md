## メモ

- `基準時点`:
  記事本文がどの時点を基準にしているか
- `公知度`:
  一般公知 / 限定公知 / 専門家公知 / 組織内限定 / 組織機密 / 国家上層限定 / 真相
- `主な認知層`:
  その情報を主に知っている集団
- `真偽`:
  通説 / 有力説 / 論争中 / 機密情報 / 真相 / 真相混在
- `変動性`:
  低 / 中 / 高
- `時点依存`:
  低 / 中 / 高
- `例外的認知者`:
  個人例外がある場合のみ記入
