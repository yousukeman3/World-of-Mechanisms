# 未使用の悪魔契約書

## 管理メタ

- 基準時点: 現代
- 公知度: 専門家公知
- 主な認知層: 呪術師 / 契約研究者 / 宗教機関の封印担当 / 闇市場業者 / 一部国家治安機関
- 真偽: 真相混在
- 変動性: 中
- 時点依存: 低

## 基本情報

- 名称: 未使用の悪魔契約書
- 別名: 空契約 / 空き魔契 / 白紙魔契
- 種別: 特異物 / 契約媒体 / 儀礼道具
- 由来: [悪魔](../../08_種族・知性体事典/精神生物・形而上生命/悪魔.md) が用いる契約媒体の未締結品
- 主用途: 契約拘束 / 呪術的誓約 / 対悪魔封印補助 / 契約研究
- 危険度: 高

## 概要

未使用の悪魔契約書とは、悪魔との契約を成立させるための媒体として作られながら、まだ特定の当事者と締結されていない状態の契約書である。見た目は空白の羊皮紙、黒い帳面、刻印片、薄い皮膜など様々だが、いずれも `契約を結ぶための枠組み` 自体が既に仕込まれている。

そのため、単なる紙ではなく、約定、代価、拘束、破棄罰則といった契約概念へ異常な親和性を持つ。対悪魔実務では危険物であると同時に、対価構造を可視化したり、封印や誓約を固定したりする高性能な道具でもある。とくに、通常の法的契約や誓約では難しい `破れにくさ` `即時執行性` `罰則の先行設定` を持たせやすいため、危険でありながら確実性の高い契約道具として重宝される。

## まず結論

- 未使用でも安全ではない
- `まだ誰のものでもない契約枠` として、極めて強い拘束力の素地を持つ
- 悪魔召喚だけでなく、封印、誓約、拘束、契約研究へ流用できる
- 不適切な記述や署名、血液接触で意図しない契約が成立する危険がある
- その代わり、実行力の高い契約、ほぼ破れない契約、違反時に即座に罰則が走る契約を比較的容易に組める

## 性質

### 契約枠の先在

- 悪魔契約書には、契約を成立させるための位相枠が既に刻まれている
- したがって、内容が未記入でも `約定が生まれやすい面` として振る舞う

### 条項定着性

- 文言、署名、血判、真名、対価記述が通常の紙より強く固定される
- 曖昧な言葉でも、意図せぬ形で拘束条項として定着することがある

### 実行力の先行付与

- この契約書上で組まれた契約は、通常の誓約や法的合意より強い `履行圧` を持つ
- 条項そのものへ執行力が宿るため、単なる約束ではなく `破れば即座に反作用が返る構造` を先に仕込める
- そのため、絶対に破れない契約、あるいは破ろうとした瞬間に契約側が先んじて罰則を執行する契約を比較的容易に設計できる

### 罰則予約

- 契約違反時の罰則をあらかじめ条項へ埋め込める
- 罰則は金銭や所有権喪失だけでなく、痛覚、拘束、記憶封印、能力停止、位置露見、真名汚染のような呪術的制裁にまで及ぶ
- したがって、未使用契約書は `契約媒体` であると同時に、条件付き執行装置でもある

### 対悪魔適合

- 悪魔の召喚、呼応、拘束に強く反応する
- そのため、悪魔を呼ぶだけでなく、逆に悪魔へ `契約を読ませる` ための檻としても使える

## 主な用途

- 対悪魔儀礼における拘束媒体
- 誓約破りを検知する呪術媒体
- 危険条項の研究用サンプル
- 契約災害の痕跡保全
- 闇市場での高額取引対象
- 高い実行力を持つ秘密契約や相互拘束契約の作成

## 危険性

- 署名、血液、真名、所有宣言で意図せぬ契約が走ることがある
- 未使用品でも、特定の悪魔系統と `相性` を持っている場合がある
- 破棄しても、契約位相が残っている限り完全には無害化されない
- 人間側が条項を雑に埋めると、相手悪魔が存在しなくても `契約されうる立場` だけが固定されることがある
- 実行力が高すぎるため、軽い気持ちで書いた条項でも取り消しにくい
- `確実な契約が組める` という利点そのものが最大の罠であり、使用者はしばしば自分で自分の退路を消す

## 法的・実務的扱い

- 多くの国家では危険儀礼物として強く規制される
- 研究や封印実務に限って保有許可が下りる場合がある
- [ISMO](../../02_組織・国家・勢力事典/軍事組織/星間治安維持機構（ISMO）.md) は悪魔を雇用しないが、越境悪魔案件の証拠保全や拘束補助として押収・封印・解析することはありうる

## 備考

- `未使用` とは、あくまで特定契約がまだ成立していないという意味であり、無垢や無害を意味しない
- 一部の高位品は、空白のままでも読む者へ `契約したい内容` を想起させる誘惑性を持つ
