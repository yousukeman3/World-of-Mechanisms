# 再現魔術士

## 管理メタ

- 基準時点: 現代
- 公知度: 一般公知
- 主な認知層: 一般市民 / 再現魔術士 / 魔導技術者 / 艦載技術者 / ISMO構成員 / 一部魔術師
- 真偽: 真相混在
- 変動性: 低
- 時点依存: 低

## 基本情報

- 名称: 再現魔術士
- 系統: 人類系 / 実務者分類 / 再現魔術運用者
- 主な成立起点: 教育 / 訓練 / 規格修得 / デバイス運用
- 身体基盤: 通常は人間またはそれに準じる観測主体の肉体
- 中核作用: 再現魔術の安定運用 / 装備接続 / 規格術式運用

## 概要

再現魔術士とは、`標準化された理論、デバイス、疑似解釈触媒、基盤を通じて再現魔術を運用する実務者` である。魔術師のように個人の内的世界や流派継承へ強く依存するというより、`既存原理への安定接続を高信頼に回す` 側に立つ。

再現魔術士の強みは、個人才能だけに依存しないことにある。規格化された術式パッケージ、演算基盤、触媒、装備、訓練体系によって、星間通信、艦載機関、個人装備、医療術式、兵装術式のような現代文明の実務を支える。

## まず結論

- 再現魔術士は `再現魔術の運用実務者` である
- 再現魔術士は存在論的分類ではなく、制度化された実務分類である
- 再現魔術士は、魔術師より `基盤依存の再現性` を重視する
- 再現魔術士は、必ずしも自力で第一質料をエーテルへ転換できなくてもよい
- ただし、エーテルを受け、流し、装備や術式へ安定接続する能力は不可欠である

## 何をしている存在か

- 標準理論と規格術式を用いて現象を安定再現する
- 個人装備、艦載機関、通信設備、医療術式、兵装術式を安全に運用する
- デバイス、疑似解釈触媒、術式演算基盤と自分自身を接続する
- 高位理論そのものを発明するより、`既に働く原理を信頼性高く回す` ことに長ける

## 成立条件

- 再現魔術の規格と理論を理解していること
- デバイス、装備、術式パッケージを適切に扱えること
- エーテルを受け、保持し、流し、安定接続できること
- 必要に応じて、疑似解釈触媒や外部基盤を含む系全体を読めること

## エーテル適性

- 星間文明圏では、再現魔術の主媒体はエーテルである
- そのため再現魔術士には、まず `エーテル運用能力` が要求される
- これはエーテルを受け、保持し、装備や術式へ安定接続する能力である
- 一方、`エーテル転換能力` はより高い技能であり、再現魔術士に常時必須ではない
- したがって再現魔術士は、純粋な魔術師ほどの自由度を持たなくても、規格化された環境では高信頼に実務できる

## 文明史上の位置

- 再現魔術が成立したことで、魔術現象は個人の資質や流派継承だけに依存せず、教育、補給、量産、互換へ乗るようになった
- その結果として再現魔術士という実務層が成立し、星間文明の広域運用を支えるようになった
- したがって再現魔術士は、単なる `便利な術者` ではなく、星間文明のインフラ運用者に近い

## 魔術師との違い

- 魔術師は `意味の側から現象を操る技術者` であり、術者本人の解釈力や内的世界へ強く依存する
- 再現魔術士は、規格化された基盤の上で再現魔術を高信頼に運用する
- 純粋な魔術師は、自力で第一質料をエーテルへ転換しつつ高い自由度で扱える場合が多い
- 再現魔術士は、必ずしもそこまでの自由度を持たない代わりに、互換性、安全性、量産性、広域実務に強い
- 差は優劣より、`個人依存の自由度` と `基盤依存の再現性` の違いにある

## 魔導技術者との違い

- 魔導技術者は、基盤、装備、演算、工廠実装、規格設計を作る側に寄る
- 再現魔術士は、その基盤を現場で運用する側に寄る
- 現実には両者を兼ねる者もいるが、役割としては `設計者` と `運用者` の差がある
- とくに ISMO や工廠では、この区別が実務上かなり重要になる

## 魔法使いとの違い

- 再現魔術士は実務分類である
- 魔法使いは `学術的に魔法へ到達した者` という到達形式である
- 再現魔術士は、高位原理へ到達していなくても成立する
- 魔法使いは理論裂開や深層理解の側へ寄るが、再現魔術士は安定運用と信頼性の側へ寄る

## 魔女との違い

- 再現魔術士は、既存原理へ接続して再現魔術を運用する
- 魔女は、世界モデルそのものを保持する側である
- 再現魔術士は制度と基盤に支えられた人間側の実務者であり、魔女とは存在論の層が違う

## 主な活躍領域

- ISMO の実働部隊、艦載運用、装備制御
- 港湾、通信、航法、工廠、医療術式運用
- 大規模インフラや星間規格に依存する現場
- ルーメンやアウレリアのような、規格化と広域運用が強い文明圏

## 注意点

- 再現魔術士は `魔術師の下位互換` ではない
- 再現魔術士は `魔導技術者` とも完全には同義ではない
- 再現魔術士は高位到達者である必要はないが、現代文明を回すうえでは不可欠である
- 現代世界では、一般市民が日常的に恩恵を受ける魔導技術の多くが、再現魔術士の層によって支えられている
