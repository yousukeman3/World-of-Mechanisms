# 魔術師

## 管理メタ

- 基準時点: 現代
- 公知度: 一般公知
- 主な認知層: 一般市民 / 魔術師 / 再現魔術士 / 魔導技術者 / 宗教者 / 深層理論研究者
- 真偽: 真相混在
- 変動性: 低
- 時点依存: 低

## 基本情報

- 名称: 魔術師
- 系統: 人類系 / 術者分類 / 技術者分類
- 主な成立起点: 才能 / 継承 / 訓練 / 体系修得
- 身体基盤: 通常は人間またはそれに準じる観測主体の肉体
- 中核作用: 意味の側から現象を操る / 術式構築 / 解釈主体の担持

## 概要

魔術師とは、`既存世界の内側で、意味の側から現象を操る技術者` である。重要なのは、魔術師がまず `技術の担い手` だという点であり、魔法使いのような高位到達や、魔女のような存在論的変質を必須としない。

魔術師は、術式、継承、流派、環境、触媒、解釈を通じて現象を成立させる。したがって魔術師の強みは、ただ出力が高いことではなく、`世界をどう読み、どの意味をどの現象として立ち上げるか` を扱えることにある。

## まず結論

- 魔術師は `意味の側から現象を操る技術者` である
- 魔術師は存在論的分類ではなく、術者分類である
- 魔術師は魔法使いや魔女と同義ではない
- 純粋魔術の担い手として語られることが多いが、核心は `既存世界の内部操作` にある
- 魔術師の熟練は、出力だけでなく `世界を読む力` と `内的世界の構築` に依存する

## 何をしている存在か

- 既存の世界の内部で、意味から現象を成立させる
- 原理、術式、解釈、魔力運用を通じて、火、治療、結界、観測、呪詛のような現象を起こす
- 必要に応じて触媒や環境を利用するが、系のどこかには必ず解釈主体が要る
- したがって魔術師は、単なるエネルギー操作員ではなく `意味を現象へ変換する術者` である

## 才能と熟練

- 魔術師の才能の中核は、`世界を読む力` と `意味へ触れる力` にある
- 現象を単なる挙動ではなく、意味を帯びたものとして把握できることが重要になる
- これに加えて、魔力を流し、留め、術式へ接続し、安定運用する能力が要る
- 高度な魔術師は、魔力の性質や位相を調整する `魔力転換` にも長ける
- 熟練は出力や技巧だけでなく、自身の内的世界をどれだけ深く構築できるかにも左右される

## 純粋魔術との関係

- `魔術師` という語は、実務上しばしば純粋魔術の担い手を強く連想させる
- これは地域伝統、継承、環境、本人の解釈力へ依存する体系で、魔術師という像が最もよく見えるからである
- ただし定義上の核心は `伝統的であること` ではなく、`既存世界の内部で意味から現象を操っていること` にある
- そのため、流派や文化圏が違っても、この条件を満たすなら広く魔術師と呼べる

## 再現魔術士との違い

- 再現魔術士は、標準化された理論、デバイス、疑似解釈触媒、基盤を通じて再現魔術を運用する実務者である
- 魔術師は、より直接に術者本人の解釈力、適性、内的世界へ依存して現象を立ち上げる
- 純粋な魔術師は、自力で第一質料をエーテルへ転換しつつ高い自由度で運用できる場合が多い
- 再現魔術士は、必ずしも自力で完全な転換ができなくても、規格化された基盤の上で高信頼に運用できればよい
- したがって差は優劣より、`個人依存の自由度` と `基盤依存の再現性` の違いにある

## 魔法使いとの違い

- 魔術師は術者分類である
- 魔法使いは `学術的に魔法へ到達した者` という到達形式である
- したがって両者の差は、単に強い弱いではなく、`魔法級の記述へ自力で到達しているか` にある
- 強力な魔術師がそのまま魔法使いとは限らず、逆に破壊出力が高くなくても魔法使いでありうる
- 魔法使いは多くの場合、世界の深層や文明の限界をより高い視座から見ている

## 魔女との違い

- 魔術師は `既存世界の内側で技術を行使する者`
- 魔女は `世界モデルそのものを保持する側へ至った存在`
- 魔術師はなお観測主体側に留まる
- 魔女は成立時点で、人格であると同時に法や世界観の担い手になっている
- このため、魔術師が世界へ働きかけるのに対し、魔女は `その世界がどう立つか` にまで食い込みうる

## 社会的な位置

- 地域社会では、伝統技芸者、術者、治療者、呪禁者、軍事要員、儀礼者として広く現れる
- 現代国家では、再現魔術体系が強い文明圏ほど、魔術師は `自由度は高いが規格外` な専門職として見られやすい
- 東天のような在地体系の厚い文明圏では、なお社会の中核技能層として強い位置を持つ
- ISMO のような規格運用組織でも、純粋な魔術師は再現魔術士と別の運用価値を持つ

## 注意点

- 魔術師は `魔法使い` の日常語的言い換えではない
- 魔術師は `魔女` の弱い版でもない
- 現代では再現魔術の普及で境界が曖昧に見えるが、深層理論では三者は別概念である
- 呪術師、祈祷師、祭儀者などの在地呼称も、深層理論上は魔術師へ含めて読める場合がある
