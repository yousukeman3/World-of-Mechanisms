# 竜種

## 管理メタ

- 基準時点: 現代
- 公知度: 高
- 主な認知層: 一般市民 / 軍人 / 狩猟者 / 魔術師 / 研究者
- 真偽: ほぼ確定
- 変動性: 低
- 時点依存: 低

## 基本情報

- 名称: 竜種
- 系統: 異種族 / 魔導生物 / 巨大捕食生物
- 主な生息圏: 山岳地帯 / 火山帯 / 極域 / 深森林 / 辺境宙域惑星
- 身体基盤: 現世肉体
- 危険度: 高

## 概要

竜種とは、端的に言えば `でかくて強い翼の生えたトカゲ` に近い、現世側の強大な魔導生物である。伝説や神話ではしばしば過剰に神秘化されるが、本質的には形而上存在ではなく、圧倒的に強靭な肉体と高い魔力適性を持つ生物種の総称として扱う方が正確である。

飛行能力、巨体、強靭な鱗や骨格、強い魔力放射、各系統ごとの特殊器官を持つため、単独で自然災害や軍事脅威に準ずる被害を出しうる。

## 性質

- 基本的には魔導生物寄りであり、自然界の捕食者や縄張り支配者として振る舞う
- 高い知能を持つ個体はいても、それは `高性能な獣` に近く、龍種のような形而上的人格とは別である
- 多くは火炎、冷気、雷、毒霧、重圧波のような生体由来の魔導器官を持つ
- 物理的な撃破や狩猟、封鎖、飼育、素材利用が原理的には可能である

## 人間社会との関係

- 辺境では災害指定種、討伐対象、あるいは信仰混じりの畏敬対象になる
- 軍事的には `生体兵器級脅威` として扱われることが多い
- 一方で、素材、血液、鱗、骨、臓器、魔力器官は高級資源として珍重される
- 民間で `人魚` と総称される存在のうち、単純に水棲の強力な魔導生物として成立しているものは竜種側へ分類される

## 龍種との違い

- [龍種](../精神生物・形而上生命/龍種.md) は形而上生物であり、竜種とは根本的に別である
- 竜種は `強い生物` だが、龍種は `挙動そのものが魔法である存在` に近い
- 龍種が竜に似た姿を取ることが多いのは、`強さ` `畏怖` `災厄` のイメージとして竜が広く共有されているためであり、竜種の上位種だからではない
- 実地では、竜を何匹か見慣れている者ほど、龍種を一目で `これは格が違う` と見抜きやすい
- 外見が似ていても、気配、存在圧、場の歪み、周囲の反応があまりに違うため、竜種の延長としては見えにくい

## 注意点

- 民間伝承では `竜` と `龍` の表記が混同されやすい
- 同様に `人魚` という通俗名も、竜種的水棲魔導生物と、龍種的高位存在の両方を雑に含みやすい
- しかし実務上は、撃破可能な大型魔導生物か、現実そのものへ権能的に触る形而上存在かで対応が全く異なる
