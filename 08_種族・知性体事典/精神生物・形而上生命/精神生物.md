# 精神生物

## 管理メタ

- 基準時点: 現代
- 公知度: 混在
- 主な認知層: 宗教者 / 魔術師 / 呪術師 / 深層理論研究者 / 一般市民
- 真偽: 真相混在
- 変動性: 低
- 時点依存: 低

## 基本情報

- 名称: 精神生物
- 系統: 形而上生命体 / アストラル系知性体
- 主な生息圏: アストラル界 / 星幽空間近傍 / 現世境界面
- 身体基盤: アストラルの肉体
- 現世顕現: エーテルやアイテールによる仮設肉体の構築
- 主な類型: [天使](./天使.md) / [悪魔](./悪魔.md)

## 概要

精神生物とは、アストラル界にアストラルの肉体を構成して存在する形而上生命体の総称である。通常は現世へ常駐せず、必要に応じてエーテルやアイテールなどの適切な魔力で現世用の肉体を組み上げ、観測主体と接触する。

魂そのものや神格そのものとは別であり、`精神や魂に近い第一質料で成立した生命` と考えた方が近い。したがって、精神生物はしばしば信仰、契約、裁定、誘惑、啓示、呪詛のような、人格主体や魂へ直接関わる局面へ現れやすい。

## 成立基盤

- 基本的な身体基盤はアストラルである
- そのため、精神、魂、願望、規範、契約のような `意味を持つ働き` と強く結びつく
- 現世では純粋なアストラル肉体を維持しにくいため、エーテルやアイテールを用いて一時的な現世肉体を構成することが多い
- 現世顕現の安定度は、供給される魔力の質、場の適性、本人の性質によって変わる

## 現世への現れ方

- 儀礼、召喚、契約、啓示、裁判、誓約、呪詛などの強い意味場へ出現しやすい
- 人型を取る個体もいれば、現象、気配、声、残響、光、影としてしか現れない個体もいる
- 現世肉体はしばしば `観測主体が理解しやすい姿` を取るが、それは本来形態そのものではない

## 観測主体・人格主体との関係

- 高度な知性と対話能力を持つ個体が多い
- ただし、人格が安定しているとは限らず、現象に近いまま振る舞う個体も少なくない
- 観測主体との関係は深いが、精神生物そのものが常に `人間の味方` であるとは限らない
- 法制度上の人格主体として扱われるかは、個体ごと、地域ごとに大きく異なる

## 主な類型

### 天使

- 具体的な秩序、美徳、正義、法の擬人化に近い
- 神々や `大機構` が敷いた則の担い手として振る舞う
- 魂と規範を中心に動く執行者であり、人間の善性へ比較的肩入れする
- 高位遺留物としては [天使の羽](../../10_博物・資源・素材事典/生体素材・精神遺留物/天使の羽.md) が重宝される

### 悪魔

- 契約を媒介に人格主体へ取りつき、対価と引き換えに利益を与える
- 人間に対して中立であり、弱さも欲望も利用対象とみなす
- 魂から魔力を摂取して繁栄する寄生的性質を持つ
- 遺留物や儀礼物としては [未使用の悪魔契約書](../../06_艦船・兵器・装備・遺物事典/古代遺物・特異物/未使用の悪魔契約書.md) が特に危険かつ有用である

## 注意点

- 精神生物は `神` と同義ではない
- 精神生物は `魔女` とも別であり、自前の世界を持ち込んで法を書き換える側ではなく、魂や規範や契約へ寄り添う形で現れやすい
- ただし、観測主体から見ると神格、怪異、死霊、妖異と混同されやすい
- とくに [妖精](../妖異・自然存在/妖精.md) や [精霊](../妖異・自然存在/精霊.md) は見た目や挙動が近く見えるが、成立基盤は自然現象と現象元素の側にあり、物質空間の魔導生物として区別される
- また [龍種](./龍種.md) のように、同じく高位で形而上的に見えても、精神生物というより `現世へ直接圧力を掛ける形而上生物` として別枠で扱うべき存在もある
