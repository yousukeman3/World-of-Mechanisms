# 魔女

## 管理メタ

- 基準時点: 現代
- 公知度: 混在
- 主な認知層: 一般市民 / 宗教者 / 魔術師 / 魔法使い / 深層理論研究者
- 真偽: 真相混在
- 変動性: 中
- 時点依存: 中

## 基本情報

- 名称: 魔女
- 系統: 高位存在 / 世界モデル保持者 / 矛盾的主体
- 主な成立起点: 観測主体由来 / 恒常機構由来 / 原初根源由来
- 身体基盤: 可変
- 現世顕現: 肉体依存とは限らない
- 中核作用: 世界観の保持 / 魔法の担持 / 法の局所的上書き

## 概要

魔女とは、単に強い魔術師や珍しい高位存在ではない。`自分の世界観そのものが主体として成立している存在` であり、魔法を `使う` というより `持っている` 側に近い。

このため魔女は、人格的個体でありながら、同時に世界モデルや法の担い手でもある。人間的に振る舞う個体も多いが、その本質は `何らかの原理が主体化したもの` に近く、成立した時点で強い矛盾を必ず孕む。

## まず結論

- 魔女は `強い術者` ではなく `世界モデルを持つ主体` である
- 魔女は魔法を技術として扱うというより、魔法を存在基盤として保持する
- したがって、魔女は `魔法使いの上位種` ではない
- 魔女は成立した時点で必ず矛盾を孕む
- ただし、原初二柱の魔女化は通常の魔女化とは意味が異なる

## 成立

- 後天的に自分の内側へ閉じた世界観を完成させ、その法を保持するに至る個体がいる
- 生来そうした世界観を当然のものとして知っている先天的個体もいる
- 世界恒常機構や高位機構が魔女として振る舞う類型もある
- いずれの場合でも共通するのは、`現象を起こす` ことより `その現象が立つ法を保持している` ことである

## 魔女とは何を持つ存在か

- 魔女は、現実へ局所的に食い込める自前の世界モデルを持つ
- そのため、現象を一つ起こすたびに既存世界の内部操作だけで済むとは限らない
- 高位の魔女ほど、原理操作と現象成立がほぼ直結して見える
- ただし理論上は、なお `原理層` と `現象層` を区別して考えた方が正確である

## 矛盾的主体

- 魔女は人格主体である
- しかし同時に、人格を超えた法や世界観の担い手でもある
- このため、`一個人である` と `一つの法である` が同居する
- ここに魔女の根本的な矛盾がある
- 原初に近い魔女ほど、この矛盾は強くなる

## `しない` と `できない`

- 魔女にとって `しない` は、単なる気分や道徳判断では終わらないことが多い
- 魔女は自分の世界観そのものを存在基盤としているため、自己定義に反する行為は `やらない` のではなく `通らない` 側へ寄りやすい
- したがって `それは私の物語ではない` `それは私の魔法ではない` `それをすると私が私でなくなる` は、実務上かなり `できない` に近い意味を持つ
- 人間からは `やろうと思えばできるのにやらない` ように見えても、魔女側では存在論的な不可能に近い場合がある
- 逆に言えば、魔女の制約は外から課される枷だけでなく、自己定義そのものから生じる

この性質のため、魔女の誓約、契約、美学、規約は、しばしば単なるポリシーではなく `その魔女がその魔女であり続けるための条件` として機能する。

## 変化と変質

- 魔女は、通常の観測主体のように `少しずつ性格や考え方が変わる` という意味では変化しにくい
- 理由は単純で、魔女にとって自己とは人格だけでなく、世界観そのものだからである
- そのため、表面的な関係性、解釈、振る舞いは変わっても、核となる判断原理は強く固定されやすい
- 逆に、その核が本当に変わるなら、それは成長や改心というより `変質` `再定義` `別の魔女としての再成立` に近い
- したがって、根本から変質することを受け入れない限り、魔女は基本的には変われない

この意味で、魔女の不変性は頑固さではなく、存在基盤の安定性に由来する。

## 通常の魔女化

- 通常の魔女化は、既存世界の上へ `その個体自身の世界観` を成立させることに近い
- これは完成済みの現実へ、別の法や見え方が食い込むことである
- そのため、通常の魔女はしばしば `自分の世界観の味方` であって、人間一般の味方とは限らない
- 魔女が全面救済装置にならないのは、能力不足よりも `何を正しい世界と見るか` が人間と一致しないためである

## 原初二柱との違い

- アルベルティーナとアイテールも魔女として語られる
- ただし原初二柱の魔女化は、通常の魔女化のように既存世界へ後から異物的な魔法を差し込むことではない
- むしろ `構造` と `意味` という当たり前そのものを、`魔法として成立しているもの` にしてしまう層に近い
- このため機構世界の通常挙動そのものが、深層ではすでに `魔法の上に乗った当たり前` になっている
- だからこそ、その当たり前へ後続の魔法や魔術が入り込む隙間が生まれる
- この意味で原初二柱は、他の魔女と連続していながら、役割としては明確に別格である

## 魔法使いとの違い

- 魔法使いは `学術的に魔法へ到達した者` という到達形式である
- 魔女は `世界モデルを保持する側へ至った存在` という存在論的分類である
- 魔法使いはなお観測主体側、人間側の枠に留まりうる
- 魔女は人格主体であっても、その成立時点で法の担い手へ踏み込んでいる
- したがって、魔法使いと魔女は重なりうるが同義ではない

## 肉体と存続

- 魔女は必ずしも通常の肉体へ強く依存しない
- 肉体を失っても、存在や魂側で成立を継続しうる類型がある
- このため、通常の死生観、刑罰観、法的処理では扱い切れない
- とくに危険なのは、存在封止が必要であっても `単に殺せば終わる` と限らない点である

## 社会と制度

- 学術的には高位存在、危険主体、例外的人格主体として整理されやすい
- 法的には、既知類型かどうか、各国法、宗教法、治安実務によって扱いが大きく異なる
- 創世聖教のように一律断罪を避けつつ、成立過程や越権性を重く見る制度もある
- 実務上は `保護` `監督` `隔離` `封止` が混ざりやすい

## 注意点

- 魔女は `女性の魔術師` を意味しない
- 魔女は `精神生物` とも `神` とも同義ではない
- 原初二柱を含むため、魔女という語は非常に広い
- ただし広いからといって曖昧なだけではなく、`世界観が主体化している` という一点では一貫している
