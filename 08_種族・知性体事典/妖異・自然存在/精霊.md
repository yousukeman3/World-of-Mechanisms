# 精霊

## 管理メタ

- 基準時点: 現代
- 公知度: 中
- 主な認知層: 一般市民 / 自然魔術師 / 呪術師 / 宗教者 / 深層理論研究者
- 真偽: 真相混在
- 変動性: 中
- 時点依存: 低

## 基本情報

- 名称: 精霊
- 系統: 魔導生物 / 妖異 / 高次環境主体
- 主な生息圏: 物質空間の地域環境 / 山岳 / 森林 / 河川流域 / 海域 / 砂漠 / 気候帯 / 都市霊場
- 成立基盤: 地域や環境に宿る意味の総体、魔力、および現象元素の結合
- 知性: 中から高
- 危険度: 中から高

## 概要

精霊とは、自然現象に宿る意味が大規模かつ高密度にまとまり、現象元素の肉体を伴って地域や環境そのものの `まとめ役` として主体化した魔導生物である。妖精より知性が高く、規模も大きく、局所環境の代弁者や管理者のように振る舞う。

神ほど共同体的・超越的ではないが、その土地や環境から見れば、きわめて重い現実的権威となる。自然魔術においては、しばしば契約相手、仲裁者、保護者、あるいは災厄の起点として扱われる。

## 妖精との違い

- [妖精](./妖精.md) が局所現象の癖や気分をまとった小主体なら、精霊は地域環境全体の整合を代表する高次主体である
- 妖精が `遊ぶ` ように魔法を使うのに対し、精霊は `環境を代行する` ように魔法を使う
- 多くの精霊は、その地域の妖精群や自然存在へ強い影響力を持つ
- どちらもアストラル生物ではなく、現象元素でできた物質空間側の存在である

## 役割

- その土地の水脈、風向、季節、繁茂、静穏、災厄の傾向をまとめて担う
- 人間社会から見ると、自然法則そのものではないが `自然現象の窓口` に近い
- 祭祀、境界管理、土地開発、災害鎮静、豊穣祈願などの局面で重要になる

## 危険性

- 魔法が使え、なおかつ高い知能を持つ存在は、それだけで通常の怪異や大型生物よりはるかに扱いづらい
- 精霊はまさにその典型であり、単なる自然災害ではなく `意思を持った環境そのもの` として振る舞いうる
- 穏当な個体なら地域の守り手になるが、敵対や激怒に転じた個体は洪水、暴風、落雷、旱魃、森林火災のような環境災害を意図的に引き起こしうる

## 龍種との重なり

- 一部には、精霊でありながら同時に [龍種](../精神生物・形而上生命/龍種.md) として顕現する個体が存在する
- そのような個体は `環境のまとめ役` であるだけでなく、`人格と権能を持つ高位災厄存在` として振る舞うため、通常の精霊よりもはるかに危険である
- 海域であれば大津波、火山帯であれば噴火、山岳域であれば崩落や気候改変のような現象を、半ば当然の権能として起こしうる
- 実務上は、自然災害・神格案件・龍種案件が重なった最悪級の例外事例として扱われる
- 通俗的に `○○龍` と呼ばれる高位存在は、この `精霊であり龍種でもある` 類型に属することが多い
- たとえば、人魚の精霊が龍種化したものは `海洋龍`、地龍の精霊が龍種化したものは `大地龍` と呼ばれやすい
- 一方で、精神性の面では `精霊の厳粛さと冷静さ` と `龍種の気性の粗さと誇り高さ` が奇妙に中和されることがあり、純粋な精霊や純粋な龍種より人間にとって話しやすい個体も少なくない
- そのため、この類型は `最悪級の災厄存在` であると同時に、十分な敬意と作法を守れば交渉可能な相手として成立する場合がある

## 人間との関係

- 敬意を払う相手であり、雑に支配できる対象ではない
- 協調すれば地域環境の安定や豊穣を得られるが、無視や乱開発は精霊災害に直結しうる
- 交渉には魔術だけでなく、祭祀、贈与、記憶、長期的信頼が重要になる

## 遺留物と素材

- 精霊の残滓や分泌物、結晶化した痕跡は高位神秘素材として珍重される
- とくに `精霊の涙` は、上級触媒や儀礼素材として重宝される一方、供給が不安定で希少である

## 注意点

- 精霊は `神` と同義ではない
- 精霊は `精神生物` とも別系統であり、アストラル界ではなく、環境と魔力と現象元素の結節から物質空間に立ち上がる
- ただし長い時間と信仰を経た精霊が、地域神や神格と境界を曖昧にする事例はある
- 同様に、高位精霊が龍種的形態と権能を取ることで、`精霊であり龍種でもある` 境界事例もありうる
