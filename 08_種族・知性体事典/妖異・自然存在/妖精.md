# 妖精

## 管理メタ

- 基準時点: 現代
- 公知度: 中
- 主な認知層: 一般市民 / 自然魔術師 / 呪術師 / 民俗学者 / 深層理論研究者
- 真偽: 真相混在
- 変動性: 中
- 時点依存: 低

## 基本情報

- 名称: 妖精
- 系統: 魔導生物 / 妖異 / 自然存在
- 主な生息圏: 物質空間の自然環境 / 森林 / 河川 / 湿地 / 廃墟 / 風穴 / 火口 / 季節境界
- 成立基盤: 自然現象の意味と魔力、および現象元素の結合
- 知性: 低から中
- 危険度: 低から中

## 概要

妖精とは、自然現象に宿る `意味` が魔力をまとい、現象元素の肉体を得て物質空間に立ち上がった魔導生物である。ISMO的な実務語で粗く言えば `自然現象の記相が主体化したもの` に近いが、実際には単なるモデルやデータではなく、現象元素から成る生きた妖異として振る舞う。

神でも天使でも悪魔でもなく、また高位の精霊とも異なる。知性は比較的低く、やんちゃで、自由で、しばしばでたらめに魔法を使う。

## 成立の仕方

- 特定の環境や現象に長く蓄積した意味が、魔力を帯びて局所的主体となる
- その主体は現象元素をまとって肉体を得るため、アストラルの仮設肉体ではなく、最初から物質空間側の存在として活動する
- したがって、同じ `火` `水` `風` に関わる妖精でも、場所や文脈が違えば性格も性質もかなり変わる
- 彼らの人格は人間のように厚く完成しているとは限らず、現象の癖や土地の気分がそのまま喋り始めたような個体も多い

## 性質

- きわめて自由で、規律や制度には従いにくい
- 善悪より `面白いか` `気に入ったか` `その場の流れに合うか` で動きやすい
- 魔法はしばしば雑で奔放だが、局所現象との噛み合いは強い
- 悪意だけで動くわけではないが、人間から見ると迷惑や災害になることは珍しくない
- 低知性ゆえに言葉が通じにくい個体も多いが、贈与、歌、遊び、儀礼にはよく反応する

## 人間との関係

- 民間伝承では、福を呼ぶ小妖、悪戯者、境界の番人として語られる
- 自然魔術の現場では、交渉相手というより `場の機嫌` に近い扱いをされることも多い
- 再現魔術工学から見ると扱いにくいが、在地理論や土地付き魔術では無視できない

## 精霊との関係

- 妖精のうち、知性と規模と一貫性が高まり、地域や環境そのもののまとめ役になったものが [精霊](./精霊.md) に近づく
- したがって、妖精と精霊は断絶した別種というより、同じ自然存在の連続体として見る方が自然である

## 注意点

- 妖精は `精神生物` とは別系統であり、アストラル界を主な基盤とせず、物質空間に現象元素の肉体を持って存在する
- 妖精は `神格` でもないため、共同体規模の信仰を直接背負うとは限らない
- ただし、長く信仰や儀礼を受けた妖精が神格や精霊へ近づく境界事例はありうる
