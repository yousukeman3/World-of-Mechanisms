# 聖職技術者制度

## 管理メタ

- 基準時点: 現代
- 公知度: 専門家公知
- 主な認知層: 聖約教国信徒 / 技術者 / 教義法研究者 / ルーメン技術官僚 / ISMO技術者
- 真偽: 通説中心
- 変動性: 中
- 時点依存: 中

## 基本情報

- 名称: 聖職技術者制度
- 公式寄り別名: 聖務技術者資格制度
- 分類: 資格制度 / 宗教技術制度 / 教義適合制度
- 所属圏: [聖約教国](../../02_組織・国家・勢力事典/国家・政府/聖約教国.md) / [創世聖教](../宗教・信仰/創世聖教.md)
- 主用途: 技術運用、教義監査、越権判定、保守、教育

## 概要

聖職技術者制度とは、[聖約教国](../../02_組織・国家・勢力事典/国家・政府/聖約教国.md) と [創世聖教](../宗教・信仰/創世聖教.md) が運用する、教義と技術の両方に通じた専門職制度である。創世聖教は技術そのものを禁じないが、越権を禁じる。そのため、単なる技術者では `どこまでが許される技術か` を裁けず、単なる聖職者では高度魔導技術を監督できない。この断絶を埋めるために成立したのが聖職技術者である。

要するに彼らは、技術を止めるための宗教者ではなく、`技術を教義の内側で安全に動かすための聖職者` である。宗教国家が反技術へ崩れずに高度文明であり続けられるのは、この制度があるからだと見てよい。

## まず結論

- 聖職技術者は `司祭 + 技術監査官 + 高度保守技師` に近い
- 技術の運用、保守、認証、越権判定を一体で担う
- 再現魔術や艦載機関は許容されるが、教義禁忌へ触れないかを彼らが見る
- 宗教国家が反技術国家にならない理由の一つが、この制度である

## 主な役割

- 再現魔術装置と艦載機関の教義適合監査
- 技術運用に関する聖務資格の付与
- 越権研究の予防審査
- 聖地、巡礼路、教区施設の保守
- 医療、救難、通信設備の適法運用
- 異端的技術や禁制研究の識別

## なぜ必要なのか

- 創世聖教は技術を全面禁止しない
- しかし [創世聖教の禁忌](../宗教・信仰/創世聖教の禁忌.md) がある以上、境界線の判断が必要になる
- 高度技術社会では、その判断を文系聖職者だけに任せることはできない
- 逆に技術者だけに任せると、教義上の停止線が失われる

そのため、教義理解と工学理解を併せ持つ専門職が制度化された。

## 教育課程

- 教義学
- 教義法と禁忌法
- 再現魔術工学基礎
- エーテル運用と安全管理
- 監査実務
- 保守実習
- 異端・越権事例研究
- 倫理と告解手続

このため、聖職技術者の育成期間は通常の聖職者や通常技術者より長くなりやすい。

## 資格階層

### 初等聖技士

- 教区設備、巡礼施設、一般機器の保守と監査補助を担う

### 聖務技師

- 艦載機関、医療設備、通信設備、封止設備の中核技師
- 越権疑義の一次判定にも関わる

### 教義技監

- 高危険設備、深層研究施設、聖地保全設備を監督する上位資格
- 教義庁や聖務監査庁と直接連携する

### 聖務工匠

- ごく少数の最高位技術聖職者
- 教義判断と工学判断の両方で大きな裁量を持つ

## 社会的な見られ方

### 信徒圏

- 頼れる技術者であり、怖い監査官でもある
- 聖職者だが、油と配線と機関にも強い人たち

### 技術者層

- 実力派も多いので一概に軽んじられない
- ただし宗教裁定が混ざるため、嫌う者もいる

### 他国

- ルーメンからは、制度としては理解できるが宗教権威が強すぎると見られやすい
- ISMOからは、案件次第で頼れるが、深層案件では厄介な相手になりうる

## 制度の効き方

- 教国圏の装置は、聖職技術者の認証なしでは正式稼働しにくい
- 巡礼路や聖地に関わる設備は、とくに彼らの権限が強い
- 高危険研究施設では、技術責任者と同格以上の発言力を持つことがある

## 問題点

- 宗教裁定と技術裁定が近すぎて、政治化しやすい
- 越権認定が保守的に寄ると、先端研究を止めすぎる
- 高位資格者の不足が慢性化しやすい
- 教義と現場の板挟みで疲弊しやすい
