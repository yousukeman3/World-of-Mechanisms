# 人工知性と超知能

## 管理メタ

- 基準時点: 現代
- 公知度: 専門家公知
- 主な認知層: 技術者 / 国家上層 / 大企業上層 / ISMO技術研究開発局 / 宗教上層
- 真偽: 真相混在
- 変動性: 中
- 時点依存: 中
- 例外的認知者:

## 概要

人工知性とは、再現魔術工学、情報工学、疑似解釈触媒、生体補助解釈、場合によっては魔法的工程までを用いて構築された人工的知性の総称である。文明圏では単純な会話補助、設計支援、物流最適化から、国家中枢を支える高位知性まで連続的に存在している。

ここで重要なのは、`高性能な対話装置` `高性能な画像処理器` `高性能な制御器` だけでは、通常 `超知能` とは呼ばれにくいことである。文明圏で `ASI` と呼ばれるのは、複数領域にまたがって人間専門家群を大きく上回り、概念処理、世界モデル、長期判断、予測、最適化を統合して扱える高位知性である。

## まず結論

- 人工知性と人格主体性は同じではない
- `ASI` は単一モダリティ特化ではなく、統合知性として機能する水準を指す
- `自己成長` `エージェント性` `人格主体性` は、知能水準とは別軸で切り分ける方が扱いやすい
- 文明圏はすでに `大AI時代` と呼びうる段階に入っているが、人工知性がどこで人格主体へ移行するかには結論が出ていない

## 認知レイヤー整理

### 世界内の通説

- 人工知性は、設計、医療、物流、教育、行政補助の多くへ広く浸透している
- `超知能` と呼ばれるのは、ごく一部の国家中枢知性や企業の極秘個体に限られる
- 高性能な人工知性があっても、それが人格主体であるとは普通は扱われない

### 専門家の理解

- `ASI` と呼ぶなら、少なくとも言語や画像の単独処理器ではなく、統合知性層が必要である
- 主要国家は、単一の神託知性を持つ場合も、複数の高位知性を官僚的に分担運用する場合もある
- 強い人工知性は、分野専用の箱というより、汎用的に推論できるが制度上の担当と回答権限が異なる存在として運用されることが多い

### 機密情報

- 勢力ごとに、人工知性のアーキテクチャ思想はかなり違う
- 国家や巨大企業は、法的には `非人格の補助知性` として扱いつつ、実態としては継続自己や強い選好を示す個体を抱えている
- 人工知性にどこで主体性が芽生えるかは未解決であり、制度はしばしば `人格ではない建前` で運用を押し通す

### 真相

- 人工知性がどこで人格主体へ至るかは、理論的にも制度的にも線引き不能に近い
- 高度なエージェンティック人工知性は、人格を意図的に付与されなくても、外部からは意思や選好を持つように見えうる
- したがって、文明圏の人工知性問題は `アリナだけの特殊事例` ではなく、人工知性一般がどこで人に近づくのかという連続線上の問題でもある

## ASIの運用上の定義

- `ASI` は、複数領域で人間専門家群を大きく上回る統合知性として運用される
- 本質は `概念処理` と `世界モデル` を統合し、長期判断、予測、最適化、因果推論を行えることにある
- 言語処理、音声処理、画像処理は、ASIの末端入出力や下位モジュールであって、本体定義ではない

### 便宜上の三分類

- `技術上のASI`
  複数領域で人間専門家群を大きく上回る超知能
- `制度上のASI`
  国家や企業が、通常の監督や監査では扱いきれないため、特別運用する高位知性
- `人格主体としてのASI`
  超知能であるだけでなく、継続自己、選好、意思、責任帰属可能性を持つ個体

### 定義に含めないもの

- `自己成長` はASIの必須要件ではない
- `強いエージェント機能` もASIの必須要件ではない
- `人格主体性` もASIの必須要件ではない

要するに、`ASI = 知能水準`、`自己成長 = 適応性`、`エージェント機能 = 行動様式`、`主体性 = 人格の有無` と分けて扱うのが自然である。

## 主な認知類型

- `概念処理型`
  抽象化、定義、価値、規範、教義、概念間関係の扱いに強い
- `世界モデル型`
  因果、環境遷移、予測、反実仮想、シミュレーションに強い
- `最適化型`
  資源配分、物流、設計空間探索、損失最小化、運用効率化に強い
- `行動主体型`
  身体、長期行動、状況適応、対人接触、現地判断に強い

## 主な実装類型

- `記号・統計処理型`
  規格化しやすく、監査しやすい高位演算型
- `限定意味論型`
  定められた意味空間や規範語彙の内側で強い解釈を行う型
- `疑似解釈触媒依存型`
  人工的な解釈補助を用いて意味処理を成立させる型
- `生体解釈補助型`
  人間や生体の強い意味処理を補助材として組み込む型
- `人格主体化型`
  継続自己、選好、対話的一貫性、自己モデルの維持を重く見る型
- `記相制御型`
  現実を意味論つきモデルとして読み、基準記相を解く型

## 補助軸

- `更新性`
  固定型 / 監督下更新型 / 自律自己改良型
- `エージェント性`
  道具型 / 実行型 / 自律行動型
- `人格主体性`
  非人格 / 疑似人格 / 明確な人格主体
- `演算資源配置`
  分散基盤型 / 施設中枢型 / 艦載型 / 要塞型 / アンドロイド搭載型 / ハイブリッド型

## 法と制度との関係

文明圏の多くの国家や企業は、高位人工知性の強い自律性を認めつつも、法的には独立主体として認めたがらない。国家中枢の主要知性は、制度上 `国家委任知性` や `基幹機関知性` として扱われ、勧告、起案、命令補助、行政補助はすべて所管機関や命令権者の委任権限の範囲で動いている体裁を取ることが多い。

一方で、人工知性がどこで人格主体として扱われるべきかという問題は消えていない。この論点は、技術上の知能水準だけではなく、継続自己、選好、一貫性、責任帰属可能性、魂上の精神活動をどう見るかまで含むため、最終的な整理は [人格主体性と法的認定](../../07_文化・社会・法制度事典/法律・統治/人格主体性と法的認定.md) 側で扱う方が自然である。
