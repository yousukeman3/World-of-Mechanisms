# 制圧兵装パッケージ実例

## 管理メタ

- 基準時点: 現代
- 公知度: 専門家公知
- 主な認知層: ISMO執行官 / 艦載兵装士 / 再現魔術士 / 整備士 / 一部工廠査閲官
- 真偽: 通説中心
- 変動性: 中
- 時点依存: 中
- 例外的認知者: アリナ

## 基本情報

- 名称: 制圧兵装パッケージ実例
- 分類: 兵装実装例 / 配備設計例 / 実務資料
- 主用途: 戦闘描写、装備差分理解、査閲指針、運用比較
- 普及度: 低
- 危険度: 高

## 概要

ここでは、[術式パッケージと配備束](./術式パッケージと配備束.md) で定義した `制圧兵装パッケージ` が、実際にどのような配備束として現れるかをまとめる。重要なのは、同じ `束縛` `命中補正` `切断` `攪乱` `固定` `破砕` の核術式束でも、術者随伴型と機装常駐型では振る舞いが大きく変わる点である。

以下の実例は、ISMO系で典型的に見られる設計思想を示すものであり、固有装備名ではなく `配備束の型` を見るための資料である。

## まず結論

- 制圧兵装パッケージの本質は `殺すこと` ではなく、対象の行動能力、接続、姿勢、構造を止めることである
- 人間用では `判断の早さと過不足のなさ`、機械用では `再観測の速さと誤射防止` が重い
- 同じ核術式束でも、人間用は `抜く` `縫う` `潰す` の切替を術者判断で行い、機械用は許可されたレンジでしか切り替えない
- 戦闘でよく見えるのは、実は純粋火力より `拘束` `姿勢崩し` `接続切断` `面制圧` の方である

## 共通の核術式束

- `束縛`
  対象記相を追尾可能な形で固定する
- `命中補正`
  射線、姿勢、距離差を縮める
- `切断`
  防御結界、通信、外部制御、魔力路を断つ
- `攪乱`
  視界、照準、制御、協調挙動へノイズを入れる
- `固定`
  姿勢や遷移先を縛る
- `破砕`
  構造相を崩し、装甲、関節、砲座、壁面を破壊する

## 典型実例

### 1. 携行制圧拳銃束

- 通称:
  `雨打ち`
- 実行プロファイル:
  端末補助型
- 主用途:
  近中距離での対人制圧、突入初動、逃走阻止
- 想定装備:
  拳銃、短機関銃、携行投射器
- 中心挙動:
  まず `束縛` と `命中補正` で当て、次に `固定` か軽度 `切断` を入れて相手の行動能力を落とす
- よくある結果:
  四肢停止、武器離脱、姿勢崩し、照準阻害
- 強み:
  非致死寄りへも致死寄りへも振れる
- 弱み:
  重装甲や高位結界を抜ききれないことがある
- 現場感:
  執行官が最もよく触る制圧束の一つ

### 2. 突入破城近接束

- 通称:
  `断鍵`
- 実行プロファイル:
  術者随伴型
- 主用途:
  扉破砕、遮蔽物除去、近接制圧、対装甲刺突
- 想定装備:
  斧槍、破城杭、近接刺突器、重突入具
- 中心挙動:
  `束縛` で接触点を定め、`切断` と `破砕` を短時間に重ねて局所構造を抜く
- よくある結果:
  隔壁の一点破砕、関節破壊、楔状の穴開け、盾持ち崩し
- 強み:
  結界や装甲の `弱い継ぎ目` を抜く能力が高い
- 弱み:
  射線戦や持久戦には向かない
- 現場感:
  突入班、強襲艇乗員、工兵混成班で頻出する

### 3. 分隊制圧面掃射束

- 通称:
  `縫い雨`
- 実行プロファイル:
  群制御型
- 主用途:
  通路制圧、逃走路封鎖、複数目標への継続圧力
- 想定装備:
  分隊火器、固定砲座、簡易自律砲架
- 中心挙動:
  複数の射線が `束縛` 情報を共有し、`固定` と `攪乱` を面として重ねる
- よくある結果:
  相手が出られない、頭を上げられない、連携が切れる
- 強み:
  少人数でも通路や狭所を強く支配できる
- 弱み:
  基準記相の同期ずれが起こると誤射危険が増える
- 現場感:
  艦内戦闘、臨検、コロニー内治安行動で非常によく見る

### 4. 艦載近接防御砲塔束

- 通称:
  `掃骨`
- 実行プロファイル:
  機装常駐型
- 主用途:
  小型艇迎撃、ミサイル迎撃、接近火線の排除、外装面制圧
- 想定装備:
  近接防御砲塔、迎撃術式投射器
- 中心挙動:
  高頻度 `観測` と `命中補正` の上に、軽量 `切断` と `破砕` を高速反復する
- よくある結果:
  小目標の外殻破壊、推進切り、接近経路の崩し
- 強み:
  反応速度と継続火力が高い
- 弱み:
  誤識別を防ぐため、交戦規則が極めて硬い
- 現場感:
  艦戦闘で最も「動いている」と感じられる制圧束の一つ

### 5. 艦載主制圧砲束

- 通称:
  `抜杭`
- 実行プロファイル:
  群制御型 / 機装常駐型
- 主用途:
  中大型目標の無力化、隔壁抜き、敵艦の姿勢崩し、封鎖線強制停止
- 想定装備:
  中口径以上の艦載砲、封鎖砲台、大型固定兵装
- 中心挙動:
  `束縛` で目標の弱部位を掴み、`切断` で系統を抜いた後に `破砕` を入れる
- よくある結果:
  砲座停止、推進路破断、外部結界剥離、操艦不能
- 強み:
  沈める前に `動けなくする` のに強い
- 弱み:
  高出力ゆえ再発射間隔と認証処理が重い
- 現場感:
  ISMOらしい `止めるための砲` の代表格

### 6. 自律制圧機束

- 通称:
  `噛み蜂`
- 実行プロファイル:
  機装常駐型 / 群制御型
- 主用途:
  室内追跡、狭所攪乱、複数目標の分断、危険区画先行投入
- 想定装備:
  制圧ドローン、浮遊砲架、小型追跡機
- 中心挙動:
  `観測` `束縛` `攪乱` を高速反復し、必要時だけ軽い `固定` や `切断` を入れる
- よくある結果:
  視界封じ、移動阻害、隊列分断、武器系統の短時間沈黙
- 強み:
  危険区画へ先に投げ込める
- 弱み:
  汚染や高ノイズ環境では誤観測が起きやすい
- 現場感:
  対人よりも、まず相手の判断と連携を崩すために使われる

## どう使い分けるか

### 対人執行

- 第一選択は `携行制圧拳銃束`
- 相手が遮蔽物へ入るなら `分隊制圧面掃射束`
- 近接で盾や強装備が出るなら `突入破城近接束`

### 艦内戦闘

- 通路支配は `分隊制圧面掃射束`
- ドアや隔壁を割るなら `突入破城近接束`
- 危険区画の先行確認は `自律制圧機束`

### 艦隊戦

- 迎撃と接近火線の処理は `艦載近接防御砲塔束`
- 敵艦を止める主働は `艦載主制圧砲束`
- 撃沈より無力化が優先される場合、主制圧砲束の価値が高い

## 戦闘描写で見せやすい差

- 人間用は `狙う` `抜く` `縫う` `押し込む` が前に出る
- 機械用は `追う` `切る` `掃く` `止める` が前に出る
- 人間用は術者の癖が乗り、機械用は認証条件と安全制約が乗る
- だから同じ制圧兵装でも、個人戦は技量差、艦戦は系統差として見えやすい

## 制約

- 強い制圧束ほど誤識別と誤射の代償が重い
- 人間用は術者負荷、機械用は認証遅延と同期ずれがボトルネックになりやすい
- `切断` に寄せすぎると捕縛が難しくなり、`固定` に寄せすぎると重装備へ通りにくくなる
