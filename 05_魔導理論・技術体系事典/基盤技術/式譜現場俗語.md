# 式譜現場俗語

## 管理メタ

- 基準時点: 現代
- 公知度: 専門家公知
- 主な認知層: 再現魔術士 / 艦載整備士 / 工廠術式管理官 / ISMO現場要員
- 真偽: 通説中心
- 変動性: 高
- 時点依存: 中
- 例外的認知者: なし

## 基本情報

- 名称: 式譜現場俗語
- 分類: 運用俗語 / 現場語彙 / 術式構成管理実務
- 主用途: 現場会話、査閲実務、配備指示、事故報告の簡略化
- 普及度: 中
- 危険度: 低

## 概要

式譜現場俗語とは、[式譜（術式構成管理系）](./式譜・術式構成管理系.md) を日常運用する現場で使われる略語、隠語、癖のある言い回しの総称である。正式文書では使われないが、工廠、艦載整備、突入班、病棟術式管理室では非常によく通る。

こうした俗語は単なる砕けた言い換えではない。高圧的な現場ほど、`どこが危ないか` `いま何をしているか` を短い語で共有する必要があるため、俗語そのものが安全文化の一部になっている。

## よく使われる言い回し

### 基本運用

- `庫を切る`
  術式庫から新しい作業写本を展開すること。
- `床に上げる`
  調律床へ変更を送ること。
- `焼く`
  刻印すること。正式には刻印だが、現場ではこちらが多い。
- `通す`
  査閲や認証を問題なく突破すること。
- `巻き戻す`
  既知の安定配備点へロールバックすること。
- `配る`
  配備束を艦や装備へ展開すること。

### 系統と差分

- `枝を切る`
  新しい系譜枝を作ること。
- `枝が太い`
  差分が多く、独自性が強くなった枝のこと。
- `枝が漂ってる`
  本線から乖離しすぎて互換性が怪しい状態。
- `本線に戻す`
  試験枝や個人枝の内容を制式系統へ合流すること。
- `差譜を見る`
  差分内容を精査すること。
- `赤い差譜`
  危険度が高い差分。特に安全制約や基準記相解決経路に触っているもの。

### 安全と事故

- `床が汚れてる`
  調律床に危険差分や未整理の変更が溜まっている状態。
- `札が飛んでる`
  認証印や権限情報が崩れている状態。
- `線が切れてる`
  監査鎖が途切れている状態。
- `黒い`
  無認証、無監査、あるいは出所不明で危険なこと。
- `焦げた`
  局所試演や本番配備で術式が暴走した、もしくは安全遮断を食らったこと。
- `遡った`
  強制遡刻や不正ロールバックを疑う言い方。

### 人間向け配備でよく言う

- `手に馴染んでる`
  個人適応ライブラリ込みで操作遅延や出力配分がよく合っていること。
- `重い`
  術者負荷が高い、感覚提示が過剰、判断待ちが多いなどで使いにくいこと。
- `噛む`
  術者の動きと端末補助のタイミングがずれていること。
- `手が勝つ`
  端末補助よりも術者本人の癖が前に出ていること。

### 機械向け配備でよく言う

- `眠りが浅い`
  常駐系が閾値に敏感すぎて、すぐ起動してしまうこと。
- `眠りが深い`
  常駐系が鈍く、起動が遅れること。
- `食う`
  触媒やエーテルを想定以上に消費すること。
- `張り付きが強い`
  固定や隔離が過剰で、解除が重いこと。
- `息が短い`
  高出力だが長時間持続できないこと。

## ISMOで特によく聞く俗語

- `アリナ寄り`
  基準記相解決や高次補正の依存が重い配備。
- `硬い束`
  安全制約が強く、現場裁量が少ない配備束。
- `柔い束`
  個人裁量や局所調整の余地が大きい配備束。
- `封じ用`
  殺傷より封鎖、拘束、時間稼ぎに寄せた術式群。
- `抜く`
  切断や無力化で相手の制御を剥がすこと。
- `縫う`
  固定や束縛で相手の動きを止めること。
- `潰す`
  破砕で構造を物理的に終わらせること。

## 俗語が多い場面

- 艦載整備区画
- 突入班の武装点検
- 工廠での配備前査閲
- 緊急事故後の巻き戻し判断
- 黒術式や汚染枝の疑いが出たとき

## 注意点

- 俗語は組織や艦ごとに差があり、完全共通ではない
- 正式な査閲文書、事故報告書、法廷提出資料では正式語へ戻す必要がある
- とくに `焼く` `巻き戻す` `遡った` のような語は、軽く言っていても実際には重い操作を指すことが多い
