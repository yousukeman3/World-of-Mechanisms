# 星間魔導インフラ

## 管理メタ

- 基準時点: 現代
- 公知度: 一般公知
- 主な認知層: 一般市民 / 海運関係者 / 加盟国政府 / ISMO構成員 / 港湾技術者
- 真偽: 通説中心
- 変動性: 中
- 時点依存: 中
- 例外的認知者:

## 基本情報

- 名称: 星間魔導インフラ
- 分類: インフラ / 共通基盤 / 星間文明維持技術
- 開発勢力: 複数文明圏 / 星間国家連合加盟圏 / 港湾・通信事業体 / ISMO
- 主用途: 通信、航法、寄港、補給、環境維持、危機対応
- 普及度: 高
- 危険度: 高

## 概要

星間魔導インフラとは、星間文明圏における通信継路、航法標識、協定港接続、検疫網、結界設備、エーテル圧管理設備をまとめた総称である。艦一隻の性能だけで星間文明が成り立っているわけではなく、`見えない標準設備の網` があるからこそ、情報、物流、補給、救難、共同軍事行動が成立する。

このインフラの存在によって、加盟圏では `情報は近いが身体は遠い` という構造が強く現れる。通信は軽い情報をエーテル継路へ載せればよいが、身体や貨物の移動には、船体、疑似解釈触媒、補給、検疫、寄港枠、航法安全が必要だからである。

## 認知状況

- 市民にとっては、星間ネット、協定港、航路、検疫体制として認識される
- 実務者にとっては、どれか一つ止まるだけでも文明機能が麻痺しかねない脆弱な相互依存網である
- 表からは地味に見えるが、戦争でも災害でも最初に奪い合われるのはこの基盤である

## 認知レイヤー整理

### 世界内の通説

- 星間通信や寄港や補給を支える共通基盤
- 協定港と共同航路を成立させる文明の土台

### 専門家の理解

- 通信継路、航法標識、環境維持、港湾接続、検疫遮断が相互接続された複合インフラである
- 目立つのは港や中継局でも、実際には背後の標準理論、疑似解釈触媒補給、監査規格が本体である
- `純機械` `魔導装置` `魔導機械` `魔導施設` の上にインフラが乗るという階層整理は、[魔導工学の実装階層](../基盤技術/魔導工学の実装階層.md) を参照

### 機密情報

- ISMOや一部国家は、非常時用の閉域継路、封鎖航路、隠し中継網を持つ
- エリミネーター案件では、通信断や航法撹乱に備えた特別プロトコルが走る

### 真相

- 文明圏の安定は、優れた艦の存在より、膨大な数の中継器、疑似解釈触媒交換、保守員、監査ログに支えられている
- したがって、派手な艦隊戦より前にインフラ疲弊が文明を殺す可能性の方が高い

## 原理

### 通信継路網

- エーテルを情報媒体として用いる広域通信継路の集合
- 情報それ自体は軽量なため、継路と中継器さえあれば星間距離でも実用的な近さを実現できる
- 逆に継路外の辺境や非加盟圏では、情報の近さが急速に失われる

### 航法標識網

- 主要航路、危険宙域、進入禁止域、寄港接続点を示す航法支援網
- 艦そのものの性能だけで全航行を賄うのではなく、外部標識と照合することで安全率を上げている
- このため標識網の破壊や偽装は、軍事行動にも海賊行為にも直結する

### 協定港接続層

- 港湾側のドック設備、認証系、検疫系、荷役系を、来訪艦の術式演算基盤と擦り合わせる接続規格
- これがあるため、加盟圏では異なる国家艦や ISMO 艦でも一定の共通手順で寄港できる
- 詳細制度は [星間航路・寄港・補給相互運用協定](../../07_文化・社会・法制度事典/条約・制度/星間航路・寄港・補給相互運用協定.md) と [協定港](../../07_文化・社会・法制度事典/条約・制度/協定港.md) に依存する

### 環境維持設備

- 気密区画、ドック、コロニー、長距離航行では、空気、水、熱、汚染、エーテル圧を一体で管理する
- ここでは `無から作る` より `既存環境を浄化・循環・安定化する` 方向が主流である
- 地味だが、実は戦闘用兵装より文明維持への寄与が大きい

### 検疫・封鎖インフラ

- 病原体、魔力汚染、精神汚染、未知物質、危険な疑似解釈触媒の流入を監視し遮断する層
- ミレイア災厄以後、この分野は特に重く扱われている
- 平時には迷惑な手続きでも、有事には文明防衛の第一線になる

## 必要資源・動力

- 大量の調律済み疑似解釈触媒
- 中継局、標識局、港湾設備
- 標準術式ライブラリと監査規格
- 保守技師、港湾労働力、監視要員
- 協定網に接続された政治的合意

## 主な用途

- 星間ネット通信
- 航路維持と安全航行
- 協定港での寄港、補給、整備
- 救難、避難、封鎖、検疫
- ISMO機動群の広域運用
- 加盟圏の共同経済圏維持

## 制約

- 非加盟圏や辺境では密度と信頼性が大きく落ちる
- 保守費が重く、政治対立で切られやすい
- 中継器、標識局、港湾基盤は破壊や偽装の標的になりやすい
- 疑似解釈触媒供給障害が起きると、通信より先に港湾や環境維持が苦しくなる場合がある

## 副作用・事故例

- 中継障害による広域通信途絶
- 航法標識の誤作動や偽装による衝突、遭難
- 港湾接続規格の不整合による寄港事故
- 汚染検知失敗による隔離破り
- 封鎖プロトコルの過剰発動による経済麻痺

## 法規制・禁忌

- 航法標識の偽装や破壊
- 協定港認証の不正取得
- 検疫ログの改竄
- 通信継路への無許可侵入
- 非常時閉域網の私的利用

## 派生技術

- 星間ネット端末
- 港湾自動荷役術式
- 航路危険予測システム
- 検疫隔離結界
- 緊急避難用の臨時中継展開装置
