# 宇宙魔術

## 管理メタ

- 基準時点: 現代
- 公知度: 混在
- 主な認知層: 宇宙ネイティブ / 外縁航宙民 / 深宇宙探査者 / 一部純粋魔術師 / 深層理論研究者 / 航宙宗教者
- 真偽: 真相混在
- 変動性: 低
- 時点依存: 低
- 例外的認知者:

## 基本情報

- 名称: 宇宙魔術
- 別名: コスモスクラフト（通称）
- 分類: 応用理論 / 自然魔術理論 / 宇宙系
- 開発勢力: 特定勢力なし
- 主用途: 宇宙全体を一つの自然として読み、その広がり、境界、因果、深淵を通じて現象へ触れる体系の整理
- 普及度: 低
- 危険度: 高

## 概要

`宇宙魔術` とは、宇宙全体を一つの自然として読み、その広がり、真空、重力井戸、放射、因果の遅れ、境界面、深淵を通じて現象へ触れる魔術体系の総称である。地上の `天象魔術` が `空` を通して上位秩序へ接続するのに対し、宇宙魔術は `空を失った後の宇宙` をそのまま自然として受け取り、そこに別種の神秘と秩序を読む。

この体系では、宇宙は単なる背景空間ではない。恒星、惑星、真空、星間物質、重力、降着円盤、事象の地平面、宇宙背景の静けさそのものまでが、自然魔術の対象になる。したがって宇宙魔術は、宇宙工学と近接しながらも、工学が扱う `観測可能な対象` を越えて、`観測の限界` や `意味が途切れる境界` まで読む体系である。

## まず結論

- 宇宙魔術は、宇宙全体を一つの自然として読む自然魔術体系である
- 地上の `天象魔術` の対になる体系だが、扱う対象は `天象` ではなく `宇宙そのもの` である
- `コスモスクラフト` は、この体系の通称や現場語として使われる
- `天体魔術` は、宇宙魔術の下位分類であり、個別天体や局所天体現象へ強く依存する流儀を指す
- 事象の地平面は、宇宙魔術において最も深い境界読解の対象の一つである
- 宇宙では神が神として見えにくいため、宇宙魔術はしばしば `神` より `深淵` `境界` `異常秩序` を読む体系になる

## なぜ宇宙魔術なのか

- 地上では、届かない `空` が神秘の面になる
- 宇宙では、その空が失われ、接触可能な外在的現実だけが残る
- そのため宇宙ネイティブにとっての神秘は、上方ではなく `外方` や `深方` に宿る
- 彼らは、無限に近い広がり、観測の遅れ、因果のねじれ、境界面、深淵を通じて宇宙を読む

したがって宇宙魔術は、`空の意味を読む体系` ではなく、`宇宙そのものの自然性を読む体系` である。

## 通称と下位分類

### コスモスクラフト

- `コスモスクラフト` は、宇宙ネイティブや深宇宙探査者のあいだで使われる俗称、通称、実務語である
- 理論名というより、宇宙を相手にする実践体系、現場感覚、流儀全体を指すことが多い
- 学術記事や制度文書では `宇宙魔術` と呼ぶ方が正確である

### 天体魔術

- `天体魔術` は、宇宙魔術の下位分類である
- 恒星、惑星、衛星、小惑星帯、降着円盤、ブラックホール、事象の地平面のような `個別天体` や `局所天体現象` を読むことに特化する
- したがって、宇宙全体の自然読解よりも、局所場、重力井戸、軌道関係、放射環境、境界面の個別性を重視する

## 何を扱う魔術か

- 宇宙魔術が主に扱うのは、次のようなものである

- 真空
- 重力井戸
- 軌道と共鳴
- 放射と静穏域
- 星間流
- 深宇宙の遅延
- 観測限界
- 事象の地平面
- `届くもの` と `届かないもの` の境界

したがって、宇宙魔術は地上自然魔術の単なる宇宙版ではない。宇宙の自然は、風土や空模様ではなく、広がりそのもの、境界そのもの、読めなくなることそのものを対象にする。

## 事象の地平面

- 宇宙魔術において `事象の地平面` は、単なる強重力境界ではない
- それは `向こう側があるのに届かない` `存在するのに読めない` `因果がこちらへ戻らない` という意味で、最も深い神秘の一つである
- 地上人が `空` に神や上位世界を見るなら、宇宙ネイティブはしばしば `地平` や `深淵` にそれを読む
- このため、事象の地平面は `深淵` `断絶` `閉じた内界` `神なき神域` といった語彙で語られやすい

## 主な技法

### 深淵読解

- 事象の地平面、重力断層、強赤方偏移域、因果遅延を読む
- 向こう側の存在そのものではなく、こちら側から観測できる限界面の挙動を扱う

### 軌道同調

- 軌道共鳴、周回周期、重力井戸の位相を利用して儀礼や術式を通しやすくする
- 天体魔術で特に重視される

### 真空聴取

- 宇宙の静穏、雑音、背景放射、空虚感を通じて異常や位相の乱れを読む
- 地上の `兆読` に近いが、意味は空ではなく真空側に宿る

### 境界固定

- 宇宙船、観測所、深宇宙施設の周辺に `ここから先は別の自然である` という境界を儀礼的に固定する
- 深宇宙での精神安定や異常侵入対策に使われる

### 深宙祈導

- 神格を直接読むというより、宇宙全体の秩序や深淵へ向けて祈りや照準を通す
- 航宙宗教や宇宙ネイティブの信仰実践と接続しやすい

## 神と深淵

- 地上では、神はしばしば `空` や `天象` を通して神として観測される
- しかし宇宙には、その神を神として見せる `空` がない
- このため、宇宙魔術はしばしば人格的な神を直接読むのではなく、`深淵` `断絶` `異常秩序` `境界` として高位現象を読む
- その結果、宇宙ネイティブの神学や自然魔術は、地上宗教よりも無機質で、深く、畏怖に寄りやすい

## 現代文明での位置づけ

### 一般社会

- 一般市民にはほとんど知られていない
- 知られていても、深宇宙航海者の迷信、外縁信仰、特殊な風習として見られやすい

### 宇宙ネイティブと外縁文化

- 宇宙暮らしが長い文化圏、深宇宙探査拠点、観測共同体、辺境航路民には比較的自然に残る
- 彼らは `空を失った後の神秘` を扱うため、天象魔術より宇宙魔術に近い感覚を持つ
- 具体的な生活圏類型としては、[宇宙ネイティブ](../../07_文化・社会・法制度事典/生活文化/宇宙ネイティブ.md)、[船団生まれ](../../07_文化・社会・法制度事典/生活文化/船団生まれ.md)、[深宇宙観測共同体](../../07_文化・社会・法制度事典/生活文化/深宇宙観測共同体.md) が近い

### 星間工学と再現魔術

- 標準理論の中心ではない
- 工学側は、真空、重力、放射、観測データを物理・魔導工学的に扱う
- 宇宙魔術は、それらを意味と境界の側から読む
- したがって、実務的に役立つ局面はあるが、標準化や機械可読化には向きにくい

## 認知レイヤー整理

### 世界内の通説

- 宇宙には地上の空とは違う、別種の怖さと神秘がある
- 深宇宙の者は独特の迷信や祈りを持つ

### 専門家の理解

- 一部の航宙宗教者、宇宙ネイティブ、純粋魔術師は、宇宙全体を自然魔術の対象として読む
- 彼らは、事象の地平面や深宇宙の境界を特別な意味の面として扱う

### 機密情報

- 深層理論研究者は、宇宙で神が `神として` 見えにくいことと、宇宙魔術が深淵や境界を読むことの対応を把握している
- 一部では、宇宙魔術を宇宙ネイティブ版の自然魔術として理論化している

### 真相

- 宇宙魔術は迷信ではない
- それは、天象を失った宇宙において、なお自然として読める層を扱う体系である
- 地上では `空` が担っていた絶対性を、宇宙では `境界` `深淵` `観測限界` が担う

## なぜ現代標準理論にならないのか

- 宇宙魔術は局所文化、生活圏、身体感覚への依存が強い
- 深宇宙ネイティブの感覚や信仰に強く結びついており、誰にでも同じ形で教えにくい
- 境界面や深淵は再現魔術の標準化対象にしにくい
- 工学が扱える部分は工学側が吸収し、残りは宗教、感覚、深層魔術へ寄る
