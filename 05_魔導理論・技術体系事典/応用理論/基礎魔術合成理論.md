# 基礎魔術合成理論

## 管理メタ

- 基準時点: 現代
- 公知度: 専門家公知
- 主な認知層: 再現魔術士 / 魔導技術者 / 一部自然魔術研究者 / 魔導教育機関
- 真偽: 真相混在
- 変動性: 低
- 時点依存: 低
- 例外的認知者:

## 基本情報

- 名称: 基礎魔術合成理論
- 分類: 応用理論 / 工学理論 / 古典的構成理論
- 開発勢力: 複数の自然魔術研究者 / 初期魔導工学者 / 近代教育機関
- 主用途: 魔術の分解と再構成、教育、基礎設計、在地体系の工学化
- 普及度: 中
- 危険度: 中

## 概要

基礎魔術合成理論とは、任意の魔術は有限個の `基本魔術` と、そのあいだをつなぐ `結合規則` の組み合わせとして理解できる、という立場を取る魔術工学理論である。

この理論は、現実を高度な意味論モデルとして読む [基準記相制御理論](./基準記相制御理論.md) より浅い層で動く。何が何者であるかを厳密に解釈するより先に、`どういう基本作用を、どう並べ、どう束ねれば、目的の現象になるか` を整理するための理論である。

とくに自然魔術や在地魔術の工学化と相性が良い。環境そのものを原理楔礎として使う自然魔術では、局地の達人芸に見える現象でも、よく観察すると `寄せる` `留める` `遮る` `移す` `変える` のような基本作用の組み合わせとして分解できることが多い。この性質を利用して、基礎教育、術式設計、標準化の入口を与えるのが本理論である。

ただし、この理論は世界の真理をそのまま写した厳密理論というより、`そう仮定すると教育と工学が回る` という強い作業仮説、あるいは工学ドグマに近い側面も持つ。ゆえに現場では便利な基礎理論として使われる一方、純粋魔術側や高位理論側からは `魔術を部品へ還元しすぎる見方` と批判されることもある。

## まず結論

- 任意の魔術は、理論上 `基本魔術` と `結合規則` の複合として記述できる
- 高度な魔術は、単一の奇跡ではなく、複数の基本作用が重なった複合術式として扱える
- したがって魔術工学の重要課題は、魔術を `新しく作る` ことだけでなく、`分解し再構成する` ことにある
- この理論は、自然魔術や在地体系を工学へ持ち込むための入口として強い
- ただし、分解できることと、再現可能であることは同義ではない
- 魔法級行為、神格介入、純粋魔術の達人芸は、きれいに分解できても工学化しにくいことがある

## 中核命題

- 魔術現象は、最小の作用単位まで分けて考えられる
- その最小作用単位を、便宜上 `基本魔術` と呼ぶ
- 基本魔術は単独でも現象を起こしうるが、実務上は複数が連接されて使われる
- 基本魔術をどの順序で、どの条件で、どの出力比でつなぐかを定める規則が `結合規則` である
- 高度な術式とは、基本魔術と結合規則を多段に組んだ `複合術式` にほかならない

## 基本魔術

- 基本魔術とは、これ以上分けずに工学上の作用単位として扱う基礎作用である
- これは世界の究極的真理という意味ではなく、教育、設計、査閲に耐える最小実務単位を指す
- 理論や流派によって切り方は変わりうるが、現代工学ではおおむね次のような分類がよく使われる

### 代表的な基本魔術

- `観測`
  対象や環境の状態を取る。
- `束縛`
  観測した対象を術式上の処理対象へ結び付ける。
- `付与`
  対象へ新たな相、属性、状態を乗せる。
- `移送`
  熱、運動、魔力、信号、粒子などを移す。
- `固定`
  状態や位置、接続を崩れにくくする。
- `遮断`
  流入、接続、干渉を切る。
- `変換`
  一つの相や資源を別の形へ変える。
- `放出`
  内部に蓄えたものを外へ出す。
- `共鳴`
  二つ以上の対象を同期、増幅、追従させる。
- `維持`
  現象を一定時間保つ。

## 結合規則

- 結合規則とは、基本魔術をどう並べ、重ね、閉じるかを定める構成規則である
- 同じ基本魔術群でも、結合規則が違えば全く別の術式になる

### 代表的な結合規則

- `直列結合`
  前段の出力を後段の入力へそのまま渡す。
- `並列結合`
  複数の基本魔術を同時に走らせる。
- `入れ子結合`
  一つの基本魔術の内部で別の複合を回す。
- `条件結合`
  観測結果や環境条件で分岐する。
- `帰還結合`
  出力を再観測して前段へ戻し、補正し続ける。
- `増幅結合`
  小さい作用を段階的に拡大する。
- `相殺結合`
  余剰出力や副作用を別経路で打ち消す。
- `封止結合`
  一定条件で停止、閉鎖、遮断する。

## 複合術式

- 複合術式とは、基本魔術と結合規則の組み合わせとして成立する術式全体である
- したがって術式設計とは、単に一つの効果名を書くことではなく、`どの基本作用を、どの順序で、どこまで閉ループ化して実行するか` を組む作業になる

### 典型例

- `結界`
  観測 -> 束縛 -> 固定 -> 遮断 -> 維持
- `治療補助`
  観測 -> 束縛 -> 変換 -> 移送 -> 維持
- `射撃補正`
  観測 -> 共鳴 -> 帰還結合 -> 固定 -> 放出
- `簡易推進補助`
  観測 -> 変換 -> 放出 -> 帰還結合

## 自然魔術との関係

- 本理論は、自然魔術を直接否定しない
- むしろ自然魔術の観察から、`基本的な作用を切り出せる` という発想が生まれたと考える方が自然である
- 自然魔術では環境そのものが原理楔礎になるため、同じ基本魔術でも局地環境によって効率や意味が変わりやすい
- このため、本理論は自然魔術の完全な代替ではなく、`自然魔術のうち工学へ持ち込める部分を抽出する理論` として働く

## 再現魔術との関係

- 再現魔術にとって本理論は、設計教育と標準化の入口として非常に便利である
- 複雑な術式を基本作用へ分解できれば、教材化、査閲、事故解析、再利用がしやすくなる
- とくに個人装備、簡易治療具、携行結界器、港湾設備のような定型分野では強い
- 一方、広域通信、艦隊統合、精密診断のような高度分野では、これだけでは対象の意味論を捉え切れず、より高位の理論が必要になる

## 基準記相制御理論との関係

- [基準記相制御理論](./基準記相制御理論.md) は、現実を `記相` という意味論モデルとして読み、比較し、制御する高位理論である
- それに対して本理論は、`どの基本作用をどう合成するか` を扱う構成理論である
- したがって両者は競合というより階層が違う
- 基礎魔術合成理論は `術式の骨格` に強く、基準記相制御理論は `対象の意味論的制御` に強い
- ISMO式のような高度体系でも、実際の術式パッケージ設計では合成理論的な切り方が下層で生きていることが多い

## 強み

- 基礎教育へ落とし込みやすい
- 在地体系を分解して比較しやすい
- 再利用、部品化、事故解析と相性が良い
- 記相のような高位概念を使わずとも、ある程度の工学設計ができる
- 自然魔術由来の体系を再現魔術へ橋渡ししやすい

## 限界

- 分解可能だからといって、同じ性能で再現できるとは限らない
- 術者の解釈力や局地環境に強く依存する魔術は、分解しても工学化しにくい
- 魔法級行為や神格介入のような上位現象は、合成だけでは説明し切れない
- 対象の意味や役割が重要な分野では、単なる作用の足し算では足りない
- 純粋魔術の達人芸は、理論上の構成図へ落ちても実演が困難なことがある

## 周辺研究

- 本理論が長く使われるにつれ、`基本魔術の合成則` そのものを形式化しようとする研究群も生まれている
- たとえば、どの結合が可換か、どの基本魔術群が閉じるか、相殺や打消しをどう記述するか、といった論点が議論される
- こうした研究はしばしば `魔術合成の代数構造` の研究として語られるが、実務現場では高級すぎるため、通常の再現魔術教育では名前だけ知られていることが多い
- ISMOや大規模研究機関では式譜査閲やライブラリ整合性へ応用されることもあるが、一般術者にとっては `学究派が深掘りしている難しい話題` という認識に留まりやすい

## 認知レイヤー整理

### 世界内の通説

- 魔術は小さい作用へ分けて考えた方が教えやすく作りやすい
- 現代の再現魔術教育でよく使われる基礎理論の一つ
- 厳密な真理というより、便利で教えやすい設計思想として受け取られている

### 専門家の理解

- この理論は万能真理ではなく、設計と教育に向いた工学的抽象化である
- 高度理論へ進む前の骨格整理として非常に有効
- 自然魔術の観察記録を工学へ翻訳する際の中間言語として強い
- 学究派の一部は、基本魔術と結合規則の形式性をさらに押し進め、代数的・構造的研究へ展開している

### 機密情報

- ISMO級の高位体系では、本理論だけでは足りず、より深い意味論理論や機構接続系が必要になる
- それでも現場実装や術式パッケージ設計の下層では、本理論的な分解が広く使われている

### 真相

- 魔術が本当に最小単位まで完全分解できるかは、世界そのものの真理としては未確定である
- ただし文明運用の水準では、`十分に役に立つ分解像` を与えるため、理論として長く生き残っている
- したがって本理論は、真理を言い当てたから残ったというより、工学ドグマとして有用だったから残った側面が強い
