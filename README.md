# 機構世界ナレッジベース

魔導工学が発展した星間文明世界「機構世界」の設定を管理するためのワークスペースです。

## 基本方針

1. `設定事典` は置かず、分野ごとの事典に分解して管理する
2. 種族ごとの生体情報は各種族ページに書く
3. 異種間交配、混血、法的扱いのような横断ルールは別資料に分ける
4. 未確定設定や矛盾確認は `90_作者用メモ` に集約する

## ディレクトリ概要

- `00_索引`: 世界観全体の導線と一覧
- `01_人物事典`: 人物ごとの個別ページ
- `02_組織・国家・勢力事典`: 国家、軍、企業、宗派などの組織情報
- `03_天体・星系・宙域事典`: 星系、惑星、航路、宙域の情報
- `04_歴史・年表`: 通史、時代区分、事件史
- `05_魔導理論・技術体系事典`: 魔導理論と各種技術の基礎情報
- `06_艦船・兵器・装備・遺物事典`: 艦船、兵器、装備、特異物の情報
- `07_文化・社会・法制度事典`: 宗教、経済、法、生活文化など
- `08_種族・知性体事典`: 種族ページと交雑・混血ルール
- `09_用語集`: 固有名詞、略語、重要概念の整理
- `90_作者用メモ`: 未確定設定、相関、整合性チェック
- `99_テンプレート`: 各事典で使うひな形

## 運用メモ

- 新しい主要項目を追加したら、必要に応じて `00_索引` の対応ページも更新する
- 種族ページには `身体構造` `寿命` `繁殖` `魔導適性` を基本項目として持たせる
- 混血や人工派生種が重要になる場合は、`08_種族・知性体事典/交雑・混血` に横断資料を置く
- 一時メモと確定設定を同じページに混ぜない

## 情報管理ルール

- 記事本文とは別に `管理メタ` を置き、`誰が知っているか` と `いつ時点の情報か` を管理する
- `誰が知っているか` はまず個人名ではなく `認知層` で管理する
- `通説` `機密情報` `真相` は本文や見出しで分ける
- `現在` とだけ書かず、必ず `基準時点` を指定する
- 状態が大きく変わる対象は、`恒常設定` と `基準時点の状態` を分ける
- 認知差の大きい記事では、本文にも `認知レイヤー整理` を置き、`世界内の通説 / 専門家の理解 / 機密情報 / 真相` を分ける
- 深層理論記事は、本文前半に通説、後半に深層理論や真相を置くと扱いやすい
