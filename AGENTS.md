# AGENTS.md

このリポジトリで作業するエージェント向けの指示。

## 基本方針

- 日本語で作業し、日本語で応答する
- 初回着手時は、まず `README.md` を確認する
- 個別の運用判断は、`README.md` から案内される関連文書を優先して確認する

## 作業開始時の確認順

1. `README.md`
2. 作業内容に応じた運用文書
   - 記事分割: `90_作者用メモ/記事分割ガイド.md`
   - 管理メタ: `99_テンプレート/記事メタテンプレート.md`
   - 時点管理: `90_作者用メモ/時点ラベル運用.md`

## 補足

- `README.md` は入口資料として扱う
- 詳細なルールは各運用文書を優先する
